    **Blind-then-debate protocol (IMP-14):**
    - Evaluator A scores all dimensions
    - Evaluator B receives transcript + A's raw scores (not explanations), commits own in separate session
    - Disagreements >10pt trigger structured exchange: B sees A's explanation, writes rebuttal
    - Tie-breaking: unresolved disputes use lower score with unreliable: true
    - Both evaluators in separate sessions (different session_id)
    - debate_enabled from contract field or --debate CLI flag
    - Capture B_score_pre_exposure for anchoring detection (P13-ADV-5-01)

    **Evaluator tier provenance (P17-CSC-03):**
    Record `evaluator_tier_used` per evaluation: DUAL/SINGLE_WITH_UNCERTAINTY/UNAVAILABLE_SENTINEL.

    **Three-tier fallback chain (P17-IMP-06):**
    (a) Primary: dual-evaluator consensus
    (b) Degraded: single-evaluator with uncertainty_flag=True, score confidence LOW
    (c) Failed: evaluation_status: unavailable sentinel
