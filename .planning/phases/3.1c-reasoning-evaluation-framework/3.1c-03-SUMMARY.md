## Decisions Made

- Re-exported ObservationSummary from transcript_parser.py rather than maintaining a parallel type in observation_parser.py. Reason: single authoritative data source (P13-IMP-08), avoids field drift between two ObservationSummary types.
- Cross-session contamination detection only runs on the glob path (no session_id). When session_id is set, the file is correct by construction (O(1) path).
- Staleness guard uses file mtime vs current time rather than session start time. This avoids needing session metadata before parsing and is simpler to test.
- max_staleness_seconds=999999999 used in real transcript tests since fixture files have old mtimes. Production default remains 3600s.
