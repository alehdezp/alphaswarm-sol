2. **DURATION-BOUNDS as separate category:** Separated from TRANSCRIPT-AUTH because duration enforcement (controller event capture) differs from transcript verification (agent hooks)
3. **TIER-C-GATING preserved as category:** Kept Tier C context gating as its own category rather than merging into EXEC-INTEGRITY because it has unique CTL-level logic and bypass markers
4. **5-step Mandatory Testing Pattern:** New structure for Agent Teams execution -- replaces the legacy 7-step pattern that referenced tmux sessions
