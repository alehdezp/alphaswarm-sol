  <action>
1. Create NEW file `src/alphaswarm_sol/queries/errors.py` with:
   - `PatternLoadError(Exception)` — base exception
   - `PatternDirectoryNotFoundError(PatternLoadError)` — takes `path: Path` in `__init__`, stores as `.path` attribute
   - `EmptyPatternStoreError(PatternLoadError)` — takes `path: Path` and `glob_pattern: str` in `__init__`
   - Include `from __future__ import annotations` and `from pathlib import Path`

2. Modify `src/alphaswarm_sol/queries/patterns.py`:
   - Add imports: `functools`, `warnings`, and the 3 exception classes from `errors.py`
   - Add module-level `_DEFAULT_VULNDOCS = Path("vulndocs")`
   - Add `_deprecated(reason)` decorator function using `functools.wraps` and `warnings.warn` with `DeprecationWarning` and `stacklevel=2`
   - Add `get_patterns(pattern_dir: Path | None = None) -> list[PatternDefinition]` function:
     - Resolves `pdir = pattern_dir or _DEFAULT_VULNDOCS`
     - Calls `PatternStore.load_vulndocs_patterns(pdir)`
     - Catches `PatternDirectoryNotFoundError` and re-raises
     - Catches `OSError` and wraps in `PatternDirectoryNotFoundError(pdir) from e`
     - Raises `EmptyPatternStoreError(pdir)` if result is empty list
   - Harden `PatternStore.__init__`: add `if not isinstance(root, Path): raise TypeError(...)`
   - Modify `PatternStore.load_vulndocs_patterns()`: add `if not vulndocs_root.exists(): raise PatternDirectoryNotFoundError(vulndocs_root)` at the start
   - Add `@_deprecated("Use get_patterns() instead. load() crashes on vulndocs index.yaml files.")` to `PatternStore.load()` and add `if not self.root.exists(): raise PatternDirectoryNotFoundError(self.root)` at the start
   - Modify `PatternEngine._load_patterns()` to be lenient: check `pdir.exists()`, log WARNING if missing and return `[]`, detect vulndocs structure, call `load_vulndocs_patterns` if found, fall back to deprecated `load()` with suppressed warning in `warnings.catch_warnings()` context

3. Update `src/alphaswarm_sol/queries/__init__.py`:
   - Add import of `get_patterns` from `alphaswarm_sol.queries.patterns`
   - Add imports of `PatternLoadError`, `PatternDirectoryNotFoundError`, `EmptyPatternStoreError` from `alphaswarm_sol.queries.errors`
   - Add all 4 to `__all__` list

**IMPORTANT:** Place `get_patterns()` AFTER the `PatternStore` class definition in patterns.py, since it calls `PatternStore.load_vulndocs_patterns()`. Place it between the `PatternStore` class and the `PatternEngine` class.
  </action>
