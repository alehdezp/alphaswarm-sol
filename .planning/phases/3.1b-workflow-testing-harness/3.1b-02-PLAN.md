**Part A — Add timing fields to ToolCall (backward-compatible):**

**Part B — Add BSKGQuery dataclass:**

```python
@dataclass
class BSKGQuery:
    command: str                    # Full alphaswarm command string
    query_type: str                 # "build-kg", "query", "pattern-query", "vulndocs"
    query_text: str                 # The query string itself
    result_snippet: str             # First 2000 chars of result
    tool_call_index: int            # Position in transcript tool_calls list
    cited_in_conclusion: bool       # Heuristic: was this result referenced later?
```

Detection heuristic: A Bash tool call is a BSKG query if `tool_input` contains an
`alphaswarm` command (`build-kg`, `query`, `vulndocs validate`, `vulndocs list`, etc.).

Parse `query_type` from the command: `build-kg` -> "build-kg", `query "..."` -> "query",
`query "pattern:..."` -> "pattern-query", `vulndocs` -> "vulndocs".

For `cited_in_conclusion`: Search subsequent assistant messages for substrings from
`result_snippet` (first 100 chars). This is a heuristic, not exact.

**Part C — Add 5 new methods to TranscriptParser:**

1. `get_bskg_queries() -> list[BSKGQuery]` — Extract all BSKG queries from tool calls
2. `graph_citation_rate() -> float` — Fraction of BSKGQuery results that were cited (0.0-1.0)
3. `get_raw_messages() -> list[dict]` — Return raw `_records` list
4. `get_message_at(index: int) -> dict` — Return single record by index
5. `get_messages_between(start_idx: int, end_idx: int) -> list[dict]` — Return slice of records

**Part D — Tests for new functionality:**

Add tests to the existing test file (or create a new section):
- Test BSKGQuery extraction from a transcript with `alphaswarm query "..."` Bash calls
- Test BSKGQuery extraction returns empty list when no alphaswarm commands
- Test query_type classification (build-kg, query, pattern-query, vulndocs)
- Test graph_citation_rate() returns 0.0 when no queries, 1.0 when all cited
- Test get_raw_messages() returns the same records as _records
- Test get_message_at() with valid and out-of-bounds index
- Test get_messages_between() returns correct slice
- Test ToolCall.timestamp field is populated from records
- **Run all existing tests and confirm they pass**

Use synthetic JSONL test data, not real transcripts. Create minimal JSONL records that
exercise the extraction logic.

```python
@dataclass
class InboxMessage:
    sender: str
    recipient: str
    content: str
    timestamp: str
    message_type: str  # "message", "broadcast", "shutdown_request", etc.

@dataclass
class AgentObservation:
    agent_id: str
    agent_type: str                      # "attacker", "defender", "verifier"
    transcript: TranscriptParser | None  # Agent's parsed transcript (None if unavailable)
    bskg_queries: list[BSKGQuery]        # Agent's graph queries
    messages_sent: list[InboxMessage]    # DMs from this agent
    messages_received: list[InboxMessage] # DMs to this agent

@dataclass
class EvidenceFlowEdge:
    from_agent: str
    to_agent: str
    evidence_id: str
    via: str  # "SendMessage", "shared_finding", etc.

@dataclass
class DebateTurn:
    agent_id: str
    agent_type: str
    position: str  # "attack", "defend", "verify"
    content_summary: str
    evidence_cited: list[str]

@dataclass
class TeamObservation:
    agents: dict[str, AgentObservation]  # agent_id -> observation
    events: EventStream | None           # Team event stream

    def get_agent_by_type(self, agent_type: str) -> AgentObservation | None:
        """Find first agent matching the given type."""
        ...

    def cross_agent_evidence_flow(self) -> list[EvidenceFlowEdge]:
        """Trace evidence passing between agents via SendMessage content."""
        ...

    def debate_turns(self) -> list[DebateTurn]:
        """Extract sequential debate turns from team messages."""
        ...

@dataclass
class CollectedOutput:
    scenario_name: str
    run_id: str
    transcript: TranscriptParser | None
    team_observation: TeamObservation | None
    structured_output: dict | None
    tool_sequence: list[str]
    bskg_queries: list[BSKGQuery]
    duration_ms: float
    cost_usd: float
    failure_notes: str = ""

@dataclass
class EvaluationGuidance:
    reasoning_questions: list[str]  # Questions for the evaluator to answer
    hooks_if_failed: list[str]      # Hook scripts to enable on re-run if scenario fails
    run_gvs: bool = True            # Run Graph Value Scoring
    run_reasoning: bool = True      # Run reasoning evaluation

class OutputCollector:
    """Aggregate scenario run results into CollectedOutput."""

    def collect(
        self,
        scenario_name: str,
        run_id: str,
        transcript: TranscriptParser | None = None,
        team_observation: TeamObservation | None = None,
        structured_output: dict | None = None,
        duration_ms: float = 0.0,
        cost_usd: float = 0.0,
        failure_notes: str = "",
    ) -> CollectedOutput:
        """Build CollectedOutput from scenario run artifacts."""
        ...

    def summary(self, output: CollectedOutput) -> str:
        """Produce human-readable summary of collected output."""
        ...
```
