  <name>Task 2: Create schema.py with label dataclasses</name>
  <files>src/true_vkg/labels/schema.py</files>
  <action>
Create schema.py with label attachment structures:

1. **LabelConfidence** enum:
   ```python
   class LabelConfidence(str, Enum):
       HIGH = "high"       # >= 0.8 confidence
       MEDIUM = "medium"   # >= 0.5 confidence
       LOW = "low"         # < 0.5 confidence

       def threshold(self) -> float:
           return {"high": 0.8, "medium": 0.5, "low": 0.0}[self.value]
   ```

2. **LabelSource** enum:
   ```python
   class LabelSource(str, Enum):
       LLM = "llm"                    # LLM-assigned
       USER_OVERRIDE = "user_override" # Manual override
       PATTERN_INFERRED = "pattern"   # Inferred from pattern match
   ```

3. **FunctionLabel** dataclass:
   ```python
   @dataclass
   class FunctionLabel:
       label_id: str                     # "access_control.owner_only"
       confidence: LabelConfidence
       source: LabelSource
       reasoning: Optional[str] = None   # Required if LOW confidence
       timestamp: datetime = field(default_factory=datetime.now)
       source_hash: Optional[str] = None # Hash of function source for staleness

       @property
       def category(self) -> str:
           return self.label_id.split(".")[0]

       @property
       def subcategory(self) -> str:
           parts = self.label_id.split(".")
           return parts[1] if len(parts) > 1 else ""

       def to_dict(self) -> Dict[str, Any]: ...

       @classmethod
       def from_dict(cls, data: Dict[str, Any]) -> "FunctionLabel": ...
   ```

4. **LabelSet** dataclass for collecting labels per function:
   ```python
   @dataclass
   class LabelSet:
       function_id: str
       labels: List[FunctionLabel] = field(default_factory=list)

       def add(self, label: FunctionLabel) -> None: ...
       def get_by_category(self, category: str) -> List[FunctionLabel]: ...
       def get_high_confidence(self) -> List[FunctionLabel]: ...
       def has_label(self, label_id: str, min_confidence: LabelConfidence = LabelConfidence.LOW) -> bool: ...
       def to_dict(self) -> Dict[str, Any]: ...

       @classmethod
       def from_dict(cls, data: Dict[str, Any]) -> "LabelSet": ...
   ```
