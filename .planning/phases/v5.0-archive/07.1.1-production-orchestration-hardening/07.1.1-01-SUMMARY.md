## Decisions Made

1. **JSONL format over SQLite** - Git-friendly, append-only, easy to diff/audit
2. **Deterministic event_id** - SHA-256 of content ensures identical events have identical IDs
3. **Optional event sourcing** - `enable_events=False` preserves backward compatibility for tests/tools that don't need events
4. **Separate work_state_updated event** - Distinct from pool_assigned to track agent state changes independently
