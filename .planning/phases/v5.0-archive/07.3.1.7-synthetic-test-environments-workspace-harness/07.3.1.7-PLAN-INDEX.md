## jj Workspace Commands Reference

| Operation | Command |
|---|---|
| Create workspace | `jj workspace add <path> [--revision <rev>]` |
| List workspaces | `jj workspace list` |
| Switch revision | `jj edit <revision>` |
| Remove workspace | `jj workspace forget <name>` |
| Show current | `jj workspace root` |
