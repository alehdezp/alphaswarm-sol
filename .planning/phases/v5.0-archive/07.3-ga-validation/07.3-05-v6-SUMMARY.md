## Key Artifacts

### scripts/test_stage_agents.sh (215 LOC)

Test script for Stage 5 agent investigations via claude-code-controller:
- Creates worktree from foundry-vault fixture
- Builds BSKG graph (prerequisite)
- Creates mock pattern matches input
- Launches Claude Code via claude-code-controller
- Requests agent investigations for reentrancy and access control findings
- Validates agent-investigations.json output
- 8-step execution flow with validation

### scripts/validate_agents.py (319 LOC)

Python validation script for agent investigation output:
- `load_investigations()` - Load from JSON with format flexibility
- `find_investigation()` - Find specific investigation by pattern/function
- `validate_investigation_exists()` - Check investigation present
- `validate_has_evidence()` - Check BSKG evidence (node_id, location, operations)
- `validate_severity()` - Check severity rating
- `validate_agent_type()` - Check correct agent used
- 8 validation checks total

### tests/fixtures/foundry-vault/expected/agent-checks.json

Expected investigation properties:
- reentrancy-classic@withdraw: critical, reentrancy-specialist
- weak-access-control@setOwner: critical, access-specialist
- Required evidence: bskg_node, code_location, exploit_path

## Test Architecture

```
Controller (test_stage_agents.sh)
  |
  +-- Step 1: Create worktree from fixture
  +-- Step 2: Build BSKG graph
  +-- Step 3: Create pattern matches input
  +-- Step 4: Launch Claude Code via claude-code-controller
  +-- Step 5: Send investigation request
  +-- Step 6: Wait for completion
  +-- Step 7: Capture transcript
  +-- Step 8: Validate with validate_agents.py
```
