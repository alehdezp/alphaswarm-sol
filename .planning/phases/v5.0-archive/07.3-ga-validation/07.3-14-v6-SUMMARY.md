## Script Capabilities

### capture_baseline.py

- **Input**: Aggregated metrics JSON from `.vrs/ga-metrics/aggregated-metrics.json`
- **Output**: Baseline JSON with:
  - Git commit, branch, date, dirty state
  - AlphaSwarm version from pyproject.toml
  - Overall precision/recall/F1 metrics
  - By vulnerability type breakdown
  - By ground truth source breakdown
  - Regression thresholds (default 5% max drop)
- **Validation**: Warns if metrics below GA targets

### regression_test.py

- **Input**: Baseline JSON + Current metrics JSON
- **Output**: Regression test report with:
  - Overall metrics comparison table
  - Per-vulnerability-type comparison
  - PASS/FAIL verdict
- **Thresholds**:
  - Overall: 5% max drop (configurable)
  - Per-type: 10% max drop

### run_baseline_capture.sh

- **Pre-checks**: Validates metrics file exists, warns if GA gate not passed
- **Execution**: Interactive prompts for overwrite confirmation
- **Output**: Prints baseline summary with key metrics

## Usage

```bash
# Capture baseline (requires aggregated metrics)
./scripts/run_baseline_capture.sh

# Or run Python script directly
uv run python scripts/capture_baseline.py \
    --metrics .vrs/ga-metrics/aggregated-metrics.json \
    --output .vrs/baselines/ga-baseline.json

# Run regression test
uv run python scripts/regression_test.py \
    --baseline .vrs/baselines/ga-baseline.json \
    --current .vrs/ga-metrics/aggregated-metrics.json \
    --output regression-report.md
```
