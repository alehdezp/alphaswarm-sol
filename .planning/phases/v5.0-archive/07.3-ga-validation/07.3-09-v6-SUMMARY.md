## Decisions Made

1. **claude-code-controller shell-first pattern** - Launch zsh first, then Claude Code, as recommended by claude-code-controller documentation to prevent losing output on errors
2. **60s idle time** - Swarm mode with multi-agent debate needs longer idle detection
3. **Preserve on failure** - Keep worktree and artifacts when test fails for debugging
4. **bc for float comparison** - Use bc -l for precision/recall threshold checks
