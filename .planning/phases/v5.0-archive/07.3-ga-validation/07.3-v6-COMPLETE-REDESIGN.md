**The product is `/vrs-audit` - a 7-stage orchestrated pipeline:**

```

Stage 1: Setup & Validation     → Install alphaswarm, verify slither
Stage 2: Build Knowledge Graph  → alphaswarm build-kg (50+ properties/function)
Stage 3: Protocol Context       → Exa MCP search for economic context
Stage 4: Pattern Matching       → 556+ patterns, Tier A/B/C
Stage 5: Agent Investigation    → Specialized agents (reentrancy, access, oracle, logic)

```

**Every test MUST validate this complete pipeline.**

### Step 8: Compare to Ground Truth
```python
import json
import yaml

# Load ground truth (EXTERNAL)
gt = yaml.safe_load((worktree / "ground-truth.yaml").read_text())

# Load findings
findings = json.loads((worktree / ".vrs/findings/verdicts.json").read_text())

# Calculate metrics
tp = fp = fn = 0
for gt_vuln in gt["vulnerabilities"]:
    found = any(f["function"] == gt_vuln["function"] for f in findings)
    if found:
        tp += 1
    else:
        fn += 1

for finding in findings:
    gt_match = any(g["function"] == finding["function"] for g in gt["vulnerabilities"])
    if not gt_match:
        fp += 1

precision = tp / (tp + fp) if (tp + fp) > 0 else 0
recall = tp / (tp + fn) if (tp + fn) > 0 else 0

print(f"Precision: {precision:.2%}, Recall: {recall:.2%}")
```

### Step 9: Cleanup
```bash
claude-code-controller kill --pane=<pane_id>
./scripts/manage_worktrees.sh cleanup wt-<scenario>
```

## FORBIDDEN PATTERNS

**Tests MUST NOT:**

```python
# ❌ Direct prompt without framework
send_prompt("Analyze this contract for vulnerabilities")

# ❌ Hardcoded findings
findings = [{"type": "reentrancy", "function": "withdraw"}]

# ❌ Mocked ground truth
ground_truth = generate_from_findings(findings)

# ❌ Perfect metrics
assert precision == 1.0 and recall == 1.0  # Suspicious

# ❌ Zero duration
assert duration > 0  # Must be real execution
```

**Tests MUST:**

```python
# ✅ Run real /vrs-audit
send_prompt("/vrs-audit contracts/")

# ✅ Parse actual output files
findings = json.loads(Path(".vrs/findings/verdicts.json").read_text())

# ✅ Use external ground truth
ground_truth = load_external_ground_truth("dvd-naive-receiver")

# ✅ Accept realistic metrics
assert 0.6 <= precision <= 0.95  # Realistic range
assert 0.5 <= recall <= 0.90     # Realistic range

# ✅ Measure real duration
assert duration > 30.0  # Real audits take time
```
