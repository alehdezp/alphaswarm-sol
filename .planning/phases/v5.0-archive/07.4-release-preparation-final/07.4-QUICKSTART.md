# AlphaSwarm.sol Private Quickstart (Secure Review)

**Goal:** Run a secure review locally in under 5 minutes without public publishing.
**Inputs:** local wheel bundle + a contract directory.
**Outputs:** a small findings pack you can hand to a reviewer.

---

## 1) Install (Private / Local Only)

### Option A: Local venv (most reliable)
```bash
python -m venv .venv
. .venv/bin/activate
pip install dist/alphaswarm_sol-*.whl
```

### Option B: pipx (isolated CLI)
```bash
pipx install dist/alphaswarm_sol-*.whl
```

**Verify:**
```bash
alphaswarm --version
alphaswarm tools status
```

---

## 2) Build a Graph (1 command)
```bash
alphaswarm build-kg examples/contracts/HelloVault.sol
```

Expected: a `.toon` graph file is created in the output folder (default or configured).

---

## 3) Run 3 Targeted Queries (Concise Review)
```bash
alphaswarm query "public functions that write state without access control"
alphaswarm query "functions with TRANSFERS_VALUE_OUT after WRITES_USER_BALANCE"
alphaswarm query "pattern:reentrancy-classic"
```

---

## 4) Export Findings Pack (1 folder)
Create a minimal findings bundle:
```
reports/
├── findings.md      # human summary
├── findings.toon    # evidence bundle
└── log.txt          # first-run trace
```

---

## 5) Failure-Mode Fast Fixes (10 lines)
- Missing tools → `alphaswarm tools status`
- No findings → run the 3 queries above against `examples/contracts/`
- Graph build fails → try `examples/contracts/HelloVault.sol`
- CLI not found → re-run install or ensure PATH includes pipx

---

## Done
You now have a local, private, repeatable secure review workflow with a compact
artifact bundle suitable for internal review and audit notes.
