# AlphaSwarm.sol v0.5.0-GA Release Notes

**Release Date:** [DATE]
**Status:** Private Release

## Overview

AlphaSwarm.sol is a behavioral vulnerability detection tool for Solidity smart contracts. Unlike name-based heuristics, it detects vulnerabilities through behavioral signatures - what code DOES, not what it's CALLED.

## Key Features

### Behavioral Signatures
- **20 semantic operations** describe code behavior abstractly
- **Operation ordering** captures dangerous patterns (e.g., transfer before state update)
- **Name-agnostic detection** works regardless of function/variable naming

### Multi-Agent Verification
- **Attacker agent** constructs exploit paths
- **Defender agent** finds guards and mitigations
- **Verifier agent** cross-checks evidence
- **Debate protocol** reduces correlated failure

### Knowledge Graph (BSKG)
- **50+ properties** per function
- **Proxy resolution** for EIP-1967, UUPS, Diamond, Beacon patterns
- **VQL 2.0 queries** for structured analysis
- **TOON format** optimized for LLM consumption

### Pattern Library
- **556+ vulnerability patterns** across 18 categories
- **Two-tier system**: Tier A (high precision) + Tier B (LLM-verified)
- **VulnDocs** unified vulnerability knowledge base

### Tool Integration
- **7 external tool adapters**: Slither, Aderyn, Mythril, Echidna, Foundry, Semgrep, Halmos
- **SARIF normalization** for unified findings
- **Smart deduplication** across tools

## Quick Start

```bash
# Install
pipx install dist/alphaswarm_sol-*.whl

# Build graph
alphaswarm build-kg path/to/Contract.sol

# Find vulnerabilities
alphaswarm query "functions with TRANSFERS_VALUE_OUT before WRITES_USER_BALANCE"
alphaswarm query "pattern:reentrancy-classic"
```

See `examples/demo/` for guided tours.
