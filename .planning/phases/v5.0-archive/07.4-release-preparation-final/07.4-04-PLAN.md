# AlphaSwarm.sol Research Paper

Technical documentation of the behavioral signature methodology for smart contract vulnerability detection.

## Quick View

The complete paper is in [PAPER.md](PAPER.md) - readable directly in any markdown viewer.

## Building PDF

Requires [pandoc](https://pandoc.org/) and a LaTeX distribution.

```bash
# Install dependencies (macOS)
brew install pandoc
brew install --cask mactex-no-gui

# Build PDF
pandoc PAPER.md -o alphaswarm-paper.pdf \
  --from markdown \
  --pdf-engine=xelatex \
  --toc \
  --number-sections \
  --bibliography=references.bib \
  --citeproc

# Build HTML
pandoc PAPER.md -o alphaswarm-paper.html \
  --from markdown \
  --standalone \
  --toc \
  --number-sections \
  --bibliography=references.bib \
  --citeproc
```
