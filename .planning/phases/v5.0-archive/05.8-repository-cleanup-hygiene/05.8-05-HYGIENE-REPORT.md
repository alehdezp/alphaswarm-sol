## Section 3: Hygiene Charter

### 3.1 Cache Taxonomy

| Cache Type | Location | Owner | Lifetime | gitignored |
|------------|----------|-------|----------|------------|
| Python cache | `__pycache__/`, `.pyc` | Python | Session | Yes |
| Pytest cache | `.pytest_cache/` | Pytest | Session | Yes |
| Ruff cache | `.ruff_cache/` | Ruff | Session | Yes |
| MyPy cache | `.mypy_cache/` | MyPy | Session | Yes |
| Build cache | `dist/`, `*.egg-info/` | Build tools | Build | Yes |
| VRS runtime | `.vrs/` (except corpus) | AlphaSwarm | Runtime | Yes |
| LLM cache | `.vrs/llm_cache/` | LLM client | Runtime | Yes |
| Crawl cache | `.vrs/crawl_cache/` | Crawler | Runtime | Yes |
| Graph cache | `.vrs/graphs/` | Builder | Runtime | Yes |

### 3.2 Required Locations for Generated Artifacts

| Artifact Type | Required Location | Notes |
|---------------|-------------------|-------|
| Knowledge graphs | `.vrs/graphs/` or output path | Runtime, gitignored |
| Test graphs | `tests/graph_cache.py` (in-memory LRU) | Not persisted |
| Benchmark corpora | `.vrs/benchmarks/corpora/` | gitignored |
| Corpus contracts | `.vrs/corpus/` | Committed |
| Planning artifacts | `.planning/phases/{phase}/` | Committed |
| Documentation | `docs/` or `vulndocs/` | Committed |
| Execution logs | `.vrs/logs/` | gitignored |
| Beads/pools | `.vrs/beads/`, `.vrs/pools/` | gitignored |

### 3.3 Suggested CI Checks

**Path Allowlist for Committed Files:**

```yaml
# Suggested CI check: Verify no runtime artifacts committed
allowed_paths:
  - src/
  - tests/ (except *.pyc, __pycache__)
  - docs/
  - vulndocs/
  - .planning/
  - .claude/
  - .github/
  - scripts/
  - validation/
  - examples/

blocked_patterns:
  - "**/__pycache__/**"
  - "**/*.pyc"
  - "**/*.pyo"
  - ".vrs/**" (except .vrs/corpus/)
  - "**/*.bak"
  - "**/*.orig"
  - "**/*.swp"
  - "**/*~"
  - ".true_vkg/**"
```

**Suggested CI jobs:**

1. **Stale reference check:** `grep -r "\.true_vkg" --include="*.py" --include="*.sh" --include="*.yaml" src/ scripts/`
2. **Backup file check:** `find . -name "*.bak" -o -name "*.orig" | grep -v .venv`
3. **Large file check:** `find . -type f -size +1M -not -path "./.git/*" -not -path "./.venv/*" -not -path "./examples/*"`

### 3.4 Soft-Delete "Attic" Concept

For future cleanups, establish an "attic" pattern before deletion:

**Protocol:**
1. Move candidates to `.planning/archive/attic/YYYY-MM-DD/`
2. Document reason and restore instructions
3. Wait 30 days (2 release cycles)
4. Delete permanently if no issues

**Example structure:**
```
.planning/archive/attic/
  2026-01-27/
    MANIFEST.md (what was moved and why)
    file1.md
    dir1/
```

**Manifest template:**
```markdown
# Attic Manifest - 2026-01-27

## Files Moved
- `file1.md`: Superseded by docs/new-file.md
- `dir1/`: One-time execution artifacts

## Restore Instructions
git checkout HEAD~N -- .planning/archive/attic/2026-01-27/file1.md

## Deletion Date
2026-02-27 (30 days from archive)
```
