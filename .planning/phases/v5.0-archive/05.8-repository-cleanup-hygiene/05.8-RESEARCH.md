## Research Plan

1. Inventory legacy content
   - List .true_vkg/ tree and file counts

2. Scan for references
   - rg -n "\.true_vkg" .
   - rg -n "crawl_cache|filtered_cache|llm_cache" .
   - rg -n "vulndocs_reference" .

3. Detect duplicates (report only)
   - Path similarity and file hash groups
   - Flag near-duplicate docs (same headings or same first 200 lines)

4. Identify unused assets
   - Find contracts and fixtures never referenced by tests
   - Identify scripts not called by docs, CLIs, or tasks

5. Produce relocation map
   - Map each legacy path to a target path or removal
