<task type="auto">
  <name>Task 2: Add economic rationality gate (EV filtering)</name>
  <files>src/alphaswarm_sol/economics/rationality_gate.py, src/alphaswarm_sol/orchestration/handlers.py</files>
  <action>
Implement economic rationality filtering based on game-theoretic analysis:

1. RationalityGate class:
   - evaluate_attack_ev(vulnerability, protocol_state) -> EVResult
   - Uses PayoffMatrix from economics/payoff.py
   - Computes: expected_profit = (success_prob * profit) - gas_cost - mev_risk

2. EVThreshold configuration:
   - filter_threshold: EV below which vulnerabilities are deprioritized
   - escalation_threshold: EV above which vulnerabilities are critical
   - Default: filter if EV < 0, escalate if EV > 10 ETH equivalent

3. filter_by_economic_rationality():
   - Filters vulnerability list by positive EV
   - Adds is_economically_rational flag to findings
   - Logs filtered vulnerabilities for transparency

4. Integration with handlers:
   - Apply rationality gate after pattern detection
   - Before spawning verifiers, check EV threshold
   - Deprioritize (not hide) economically irrational vulnerabilities
  </action>

