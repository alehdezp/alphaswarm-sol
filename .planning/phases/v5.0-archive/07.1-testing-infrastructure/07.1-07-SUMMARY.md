## Test Patterns Used

All tests follow these patterns per plan requirements:

1. **Mock Dependencies** - All external dependencies (merger, verifier, bead_factory, LLM client) are mocked
2. **Deterministic Fixtures** - Predefined test data ensures reproducible results
3. **Component Isolation** - Each agent tested independently
4. **No Direct LLM Calls** - Tests verify agents delegate to runtime, don't execute directly
