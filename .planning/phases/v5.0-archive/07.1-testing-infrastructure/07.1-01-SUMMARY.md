## Decisions Made

1. **Model assignments optimized for task complexity:**
   - Opus 4 for test-conductor (complex coordination) and gap-finder (complex reasoning)
   - Sonnet 4 for corpus-curator (balanced validation) and regression-hunter (pattern matching)
   - Haiku 4 for mutation-tester (mechanical generation) and benchmark-runner (metric collection)

2. **Conductor delegation pattern:** Test conductor orchestrates but never executes tests directly - all work delegated to specialized agents

3. **EMA ranking updates:** Benchmark runner uses exponential moving average (alpha=0.3) to update model rankings, preventing single-run outliers from distorting scores

4. **Mutation validation strategy:** Each mutation validated via semantic equivalence check (BSKG operation comparison) and AST structural comparison
