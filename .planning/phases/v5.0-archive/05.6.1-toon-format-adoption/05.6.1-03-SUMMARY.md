## Technical Details

### TOON File Structure (After Fix)
```toon
format: alphaswarm-kg-v1
saved_at: "2026-01-23T18:20:26.408881+00:00"
graph_json: "{\"metadata\":{},\"nodes\":[...],\"edges\":[...]}"
```

The graph_json field contains the full graph as a compact JSON string, avoiding the toons library bug while still benefiting from TOON's simpler envelope format.
