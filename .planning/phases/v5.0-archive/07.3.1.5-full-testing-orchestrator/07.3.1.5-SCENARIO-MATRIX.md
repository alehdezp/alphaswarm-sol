# 07.3.1.5 Scenario Matrix (95 Real-World Situations)

All scenarios MUST be executed via Claude Code using claude-code-controller in `alphaswarm-claude-code-agent-teams-session`. Each scenario uses a worktree and stage snapshots (graph, context, patterns, agents, debate, report).

**Full claude-code-controller documentation:** See `.planning/testing/rules/claude-code-controller-REFERENCE.md` and `.planning/testing/rules/canonical/claude-code-controller-instructions.md`

## Scenario Format

```
ID - Scenario - Focus - Ground Truth Source - Required /vrs commands
```

**Ground Truth Sources (Valid):**
- DVD: Damn Vulnerable DeFi (https://github.com/tinchoabbate/damn-vulnerable-defi)
- SmartBugs: SmartBugs Curated Dataset (https://github.com/smartbugs/smartbugs-curated)
- SWC: Smart Contract Weakness Classification (https://swcregistry.io)
- Audit: Published security audit reports
- Internal: AlphaSwarm test corpus (tests/contracts/)
