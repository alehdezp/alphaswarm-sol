- [ ] **E010: claude-code-agent-teams session crash** - Session died unexpectedly
- [ ] **E011: Evidence pack write failure** - Could not write artifacts
- [ ] **E015: Snapshot checksum mismatch** - Data corruption detected

### Detection Failures
- [ ] **E006: False negative** - Expected finding not detected
- [ ] **E012: False positive** - Unexpected finding reported
- [ ] **E007: Ground truth missing** - No baseline for comparison

### Debug/Validation Failures
- [ ] **E013: Missing debug record** - Skill/agent did not emit debug output

## Quick Reference: Completing This Template

### Required Sections (minimum viable postmortem)

1. **Summary** - What happened
2. **Failure Category** - Check exactly one box
3. **Evidence Pack Links** - Where is the evidence
4. **Reproduction Steps** - How to trigger it
5. **Root Cause Analysis** - Why it happened
6. **Fix Proposal** - How to fix it
7. **Sign-Off** - Verify fix works

### Optional Sections (for thorough analysis)

- Prevention (7)
- Lessons Learned (8)
- Appendices (A, B, C)

### Time Budget

| Activity | Max Time |
|----------|----------|
| Initial triage | 30 min |
| Root cause investigation | 2 hours |
| Fix implementation | 2 hours |
| Verification | 1 hour |
| Documentation | 30 min |
| **Total** | **6 hours max** |

If root cause not found in 2 hours, escalate and document unknowns.
