## Decisions Made

1. **10 QA Gates Structure:**
   - G01-G07: Blocking gates (stage completion, precision, recall, F1, debug coverage, Tier-C, economic context)
   - G08: Warning-only gate (agent agreement >= 0.90)
   - G09-G10: Infrastructure gates (transcript validity, duration validity)

2. **GA Thresholds:**
   - Precision >= 0.85 (minimize false positives)
   - Recall >= 0.80 (minimize false negatives)
   - F1 >= 0.82 (balanced measure)
   - Duration > 5000ms (prevent mocked runs)

3. **Gate Evaluation Order:**
   - Transcript validity first (early exit on invalid pack)
   - Duration validity second (catch mocked runs)
   - Debug coverage third (observability required)
   - Then accuracy metrics

4. **Bootstrap Detection:**
   - Single known-vuln scenario (S01: DVD Naive Receiver) must pass before full suites
   - Prevents wasting time on full runs when basic detection is broken
