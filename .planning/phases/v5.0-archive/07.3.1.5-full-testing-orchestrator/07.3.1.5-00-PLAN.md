## Strict Quality Controls
- Enforce claude-code-controller usage as a hard gate; missing markers invalidate the run.
- Validate debug-mode artifacts for every skill/agent per the debug matrix.
- Evidence packs are mandatory inputs to gate evaluation; missing packs fail the run.

## Iteration Protocol (max 10, real runs only)
- Run up to 10 real iterations per phase: run → evidence pack → gates → failure classification → fix → re-run.
- Any missing evidence pack artifacts, missing claude-code-agent-teams markers, or duration <= 5000ms is an automatic fail.
- No simulated results, no fabricated metrics, no mock detections; only real claude-code-agent-teams-driven runs count.
- After 10 failed iterations, stop and produce a postmortem with evidence pack links and remediation plan.
