### 3. Human Workflow Runner

Located at `scripts/e2e/run_human_workflows.sh`.

CLI features:

```bash
# List all workflows (dry-run)
./scripts/e2e/run_human_workflows.sh --dry-run

# Run specific workflow
./scripts/e2e/run_human_workflows.sh --workflow HAW-004 --debug

# Resume from snapshot
./scripts/e2e/run_human_workflows.sh --resume HAW-001

# Validate workflow YAML
./scripts/e2e/run_human_workflows.sh --validate
```

Key capabilities:
- claude-code-controller enforcement for all execution
- Worktree isolation per workflow
- Snapshot create/verify/restore with checksums
- Evidence pack creation per workflow
- Missing snapshot on resume = failure (exit code 5)
