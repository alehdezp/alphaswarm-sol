## Decisions Made

1. **claude-code-controller usage is a hard gate** - No bypass, no exceptions; all test runs must use claude-code-agent-teams sessions
2. **5000ms minimum duration** - Prevents mock/simulated execution from passing validation
3. **Socket isolation required** - Each run must use unique socket file for parallel safety
4. **Prohibited pattern detection** - Scans for subprocess.run, os.system, etc. to catch bypass attempts
