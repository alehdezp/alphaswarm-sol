# /vrs-full-testing Execution Playbook

This playbook provides step-by-step instructions for running `/vrs-full-testing` in production. Operators MUST follow this playbook exactly to ensure valid evidence packs and reproducible results.

---

| claude-code-agent-teams | >= 3.0 | `claude-code-agent-teams -V` |

## Phase 1: Environment Setup

### Step 1.1: Create Isolated claude-code-agent-teams Session

```bash
# Kill any existing test session
claude-code-agent-teams kill-session -t alphaswarm-test 2>/dev/null || true

# Create fresh session with logging enabled
claude-code-agent-teams new-session -d -s alphaswarm-test -x 200 -y 50

# Configure session for transcript capture
claude-code-agent-teams set-option -t alphaswarm-test history-limit 50000

# Verify session is running
claude-code-agent-teams has-session -t alphaswarm-test && echo "Session ready" || echo "ERROR: Session creation failed"
```

### Step 1.2: Create Git Worktree for Isolation

```bash
# Generate unique run ID
RUN_ID="run-$(date +%Y%m%d)-$(openssl rand -hex 4)"
echo "Run ID: $RUN_ID"

# Create worktree directory
WORKTREE_PATH="/tmp/alphaswarm-test-${RUN_ID}"
git worktree add "$WORKTREE_PATH" HEAD --detach

# Verify worktree is functional
cd "$WORKTREE_PATH"
git status && echo "Worktree ready: $WORKTREE_PATH"

# Store paths for later use
export ALPHASWARM_RUN_ID="$RUN_ID"
export ALPHASWARM_WORKTREE="$WORKTREE_PATH"
export ALPHASWARM_EVIDENCE="/tmp/evidence-pack-${RUN_ID}"
```

### Step 1.3: Create Evidence Pack Directory

```bash
# Create evidence pack structure
mkdir -p "$ALPHASWARM_EVIDENCE"/{transcripts/{suite-s1,suite-s2,suite-s3,suite-s4},snapshots,debug/{skills,agents},metrics,artifacts/{graphs,context,patterns,reports,debates},ground-truth}

# Initialize manifest
cat > "$ALPHASWARM_EVIDENCE/manifest.json" << EOF
{
  "\$schema": "https://alphaswarm.dev/schemas/evidence-pack-v1.json",
  "version": "1.0.0",
  "run_id": "$ALPHASWARM_RUN_ID",
  "created_at": "$(date -u +%Y-%m-%dT%H:%M:%S.000Z)",
  "worktree": {
    "path": "$ALPHASWARM_WORKTREE",
    "created_at": "$(date -u +%Y-%m-%dT%H:%M:%S.000Z)"
  },
  "claude-code-agent-teams": {
    "session_name": "alphaswarm-test"
  }
}
EOF

echo "Evidence pack initialized: $ALPHASWARM_EVIDENCE"
```

## Phase 2: Run /vrs-full-testing

### Step 2.1: Execute in Record Mode

Record mode captures transcripts and creates baseline for future diff comparisons.

```bash
# Switch to worktree
cd "$ALPHASWARM_WORKTREE"

# Install project in editable mode within worktree
uv pip install -e .

# Run full-testing with all scenarios
# Record mode creates baseline transcripts
/vrs-full-testing --mode record --scenarios S01-S72 --evidence "$ALPHASWARM_EVIDENCE"
```

**Expected Duration:** 60-90 minutes for full 72-scenario run.

**Recovery Path (if interrupted):**
```bash
# Check last successful scenario
cat "$ALPHASWARM_EVIDENCE/manifest.json" | jq '.scenarios[-1].id'

# Resume from next scenario
/vrs-full-testing --mode record --scenarios S${LAST_COMPLETED+1}-S72 --evidence "$ALPHASWARM_EVIDENCE" --resume
```

### Step 2.2: Execute in Diff Mode (Regression Testing)

Diff mode compares current run against golden transcripts.

```bash
# Run in diff mode (requires baseline from previous record run)
/vrs-full-testing --mode diff --baseline /path/to/baseline-evidence-pack --evidence "$ALPHASWARM_EVIDENCE"
```

**Diff Output Interpretation:**
| Status | Meaning | Action |
|--------|---------|--------|
| MATCH | Transcript matches baseline | Continue |
| TIMING_DIFF | Timing changed, content same | Warning, continue |
| CONTENT_DIFF | Output changed | FAIL, investigate |
| MISSING | Expected artifact not found | FAIL, investigate |

## Phase 3: Snapshot Management

### Step 3.1: Create Suite Snapshots

After each suite completes, capture worktree state.

```bash
# After Suite 1 (Command Tests)
tar -czf "$ALPHASWARM_EVIDENCE/snapshots/s1-snapshot.tar.gz" \
    -C "$ALPHASWARM_WORKTREE" \
    .vrs/ tests/contracts/ vulndocs/ \
    --exclude="*.pyc" --exclude="__pycache__"

# Verify snapshot integrity
tar -tzf "$ALPHASWARM_EVIDENCE/snapshots/s1-snapshot.tar.gz" | head -5

# Record checksum
sha256sum "$ALPHASWARM_EVIDENCE/snapshots/s1-snapshot.tar.gz" >> "$ALPHASWARM_EVIDENCE/snapshots/checksums.txt"
```

### Step 3.2: Restore from Snapshot (Recovery)

```bash
# Verify checksum before restore
EXPECTED=$(grep "s1-snapshot.tar.gz" "$ALPHASWARM_EVIDENCE/snapshots/checksums.txt" | cut -d' ' -f1)
ACTUAL=$(sha256sum "$ALPHASWARM_EVIDENCE/snapshots/s1-snapshot.tar.gz" | cut -d' ' -f1)

if [ "$EXPECTED" != "$ACTUAL" ]; then
    echo "ERROR: Snapshot checksum mismatch - E015"
    exit 1
fi

# Restore snapshot
cd "$ALPHASWARM_WORKTREE"
tar -xzf "$ALPHASWARM_EVIDENCE/snapshots/s1-snapshot.tar.gz"
echo "Snapshot restored successfully"
```

## Phase 4: Transcript Collection

### Step 4.1: Capture claude-code-agent-teams Transcripts

Each scenario must produce a marked transcript.

```bash
# Before scenario execution
SCENARIO_ID="S01"
START_MARKER="[ALPHASWARM-START] $SCENARIO_ID $(date -u +%Y-%m-%dT%H:%M:%S.000Z)"
claude-code-agent-teams send-keys -t alphaswarm-test "echo '$START_MARKER'" Enter

# After scenario execution
EXIT_CODE=$?
DURATION_MS=$(( ($(date +%s) - $SCENARIO_START_EPOCH) * 1000 ))
END_MARKER="[ALPHASWARM-END] $SCENARIO_ID $EXIT_CODE $DURATION_MS"
claude-code-agent-teams send-keys -t alphaswarm-test "echo '$END_MARKER'" Enter

# Capture transcript
claude-code-agent-teams capture-pane -t alphaswarm-test -p -S -50000 > "$ALPHASWARM_EVIDENCE/transcripts/suite-s1/$SCENARIO_ID.txt"
```

### Step 4.2: Validate Transcript Markers

```bash
# Run transcript validation
uv run python scripts/e2e/validate_claude_code_controller_usage.py "$ALPHASWARM_EVIDENCE/transcripts/"

# Expected output:
# Transcripts validated: 72/72
# Markers present: 72/72
# Durations valid (>5000ms): 72/72
```

## Phase 6: Gate Evaluation

### Step 6.1: Run Gate Checks

```bash
# Evaluate GA gates using evidence pack
uv run python -c "
from pathlib import Path
import json

pack_path = Path('$ALPHASWARM_EVIDENCE')
accuracy = json.load(open(pack_path / 'metrics/accuracy.json'))

# Gate thresholds
PRECISION_MIN = 0.85
RECALL_MIN = 0.80
F1_MIN = 0.82

precision = accuracy['overall']['precision']
recall = accuracy['overall']['recall']
f1 = accuracy['overall']['f1_score']

print(f'Precision: {precision:.2%} (min: {PRECISION_MIN:.0%}) - {\"PASS\" if precision >= PRECISION_MIN else \"FAIL\"}')
print(f'Recall: {recall:.2%} (min: {RECALL_MIN:.0%}) - {\"PASS\" if recall >= RECALL_MIN else \"FAIL\"}')
print(f'F1 Score: {f1:.2%} (min: {F1_MIN:.0%}) - {\"PASS\" if f1 >= F1_MIN else \"FAIL\"}')

if precision >= PRECISION_MIN and recall >= RECALL_MIN and f1 >= F1_MIN:
    print('\\nGATE: PASS')
else:
    print('\\nGATE: FAIL')
"
```

### Step 6.2: Generate Gate Report

```bash
# Generate final gate report
uv run alphaswarm gate-report "$ALPHASWARM_EVIDENCE" --output "$ALPHASWARM_EVIDENCE/gate-report.md"
```

```bash
# Archive evidence pack
```

```bash
# Only cleanup after successful run and archive
if [ -f ".vrs/evidence-archive/${ALPHASWARM_RUN_ID}.tar.gz" ]; then
    git worktree remove "$ALPHASWARM_WORKTREE" --force
    echo "Worktree removed"
else
    echo "WARNING: Preserving worktree - archive not confirmed"
fi
```

### Step 7.3: Kill claude-code-agent-teams Session

```bash
# Preserve session on failure, kill on success
if [ "$GATE_STATUS" = "PASS" ]; then
    claude-code-agent-teams kill-session -t alphaswarm-test
    echo "claude-code-agent-teams session killed"
else
    echo "Preserving claude-code-agent-teams session for debugging: claude-code-agent-teams attach -t alphaswarm-test"
fi
```

## Recovery Procedures

### Recovery: claude-code-agent-teams Session Crash

```bash
# Check if session exists
claude-code-agent-teams has-session -t alphaswarm-test 2>/dev/null || {
    echo "ERROR: claude-code-agent-teams session crashed (E010)"

    # Preserve worktree and evidence
    echo "Worktree preserved at: $ALPHASWARM_WORKTREE"
    echo "Evidence preserved at: $ALPHASWARM_EVIDENCE"

    # Create postmortem
    cat > "$ALPHASWARM_EVIDENCE/postmortem-$(date +%Y%m%d-%H%M%S).md" << 'POSTMORTEM'
## claude-code-agent-teams Session Crash Postmortem

**Run ID:** $ALPHASWARM_RUN_ID
**Timestamp:** $(date -u +%Y-%m-%dT%H:%M:%SZ)

### Failure Category
Infrastructure failure (E010)

### Evidence Pack
$ALPHASWARM_EVIDENCE

### Next Steps
1. Check system logs for claude-code-agent-teams crash cause
2. Verify evidence pack integrity
3. Resume from last successful scenario
POSTMORTEM

    exit 1
}
```

### Recovery: Worktree Corruption

```bash
# Verify worktree is functional
cd "$ALPHASWARM_WORKTREE" && git status || {
    echo "ERROR: Worktree corrupted"

    # Attempt recovery from latest snapshot
    LATEST_SNAPSHOT=$(ls -t "$ALPHASWARM_EVIDENCE/snapshots/"*.tar.gz 2>/dev/null | head -1)

    if [ -n "$LATEST_SNAPSHOT" ]; then
        # Recreate worktree
        git worktree remove "$ALPHASWARM_WORKTREE" --force 2>/dev/null
        git worktree add "$ALPHASWARM_WORKTREE" HEAD --detach

        # Restore from snapshot
        cd "$ALPHASWARM_WORKTREE"
        tar -xzf "$LATEST_SNAPSHOT"
        echo "Recovered from: $LATEST_SNAPSHOT"
    else
        echo "No snapshots available - full restart required"
        exit 1
    fi
}
```

### Recovery: Evidence Pack Write Failure

```bash
# Test evidence pack writability
touch "$ALPHASWARM_EVIDENCE/test-write" 2>/dev/null || {
    echo "ERROR: Cannot write to evidence pack (E011)"

    # Attempt to create alternative location
    ALT_EVIDENCE="$HOME/.alphaswarm-evidence-${ALPHASWARM_RUN_ID}"
    mkdir -p "$ALT_EVIDENCE"

    # Copy what we have
    cp -r "$ALPHASWARM_EVIDENCE"/* "$ALT_EVIDENCE"/ 2>/dev/null

    export ALPHASWARM_EVIDENCE="$ALT_EVIDENCE"
    echo "Switched to alternative evidence location: $ALT_EVIDENCE"
}
rm -f "$ALPHASWARM_EVIDENCE/test-write"
```

## Quick Reference

### Environment Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `ALPHASWARM_RUN_ID` | Unique run identifier | `run-20260130-a1b2c3d4` |
| `ALPHASWARM_WORKTREE` | Isolated worktree path | `/tmp/alphaswarm-test-run-20260130-a1b2c3d4` |
| `ALPHASWARM_EVIDENCE` | Evidence pack directory | `/tmp/evidence-pack-run-20260130-a1b2c3d4` |
| `ALPHASWARM_DEBUG` | Enable debug mode | `true` |

### Key Commands

| Action | Command |
|--------|---------|
| Start run | `/vrs-full-testing --mode record` |
| Resume run | `/vrs-full-testing --mode record --resume` |
| Diff run | `/vrs-full-testing --mode diff --baseline <path>` |
| Collect evidence | `scripts/e2e/collect_evidence_pack.sh <evidence_dir> <worktree>` |
| Validate pack | `uv run python scripts/e2e/validate_evidence_pack.py <evidence_dir>` |

### Error Codes

| Code | Meaning | Recovery |
|------|---------|----------|
| E003 | Missing START marker | Re-run scenario with proper claude-code-agent-teams wrapper |
| E004 | Missing END marker | Check for scenario crash, re-run |
| E005 | Duration too short (<5000ms) | Scenario did not run properly, re-run |
| E010 | claude-code-agent-teams session crash | Restart session, resume from last scenario |
| E011 | Evidence pack write failure | Switch to alternative evidence location |
| E013 | Missing debug record | Re-run with DEBUG_MODE=true |
| E015 | Snapshot checksum mismatch | Recreate snapshot from clean state |
