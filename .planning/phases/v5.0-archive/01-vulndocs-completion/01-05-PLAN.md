<task type="auto">
  <name>Task 2: Execute pattern deduplication and merge</name>
  <files>knowledge/vulndocs/categories/**/*.md</files>
  <action>
    Process each duplicate candidate group from the inventory:

    **Deduplication rules (from CONTEXT.md):**
    1. Same semantic signature = MERGE regardless of surface differences
    2. Higher authority source wins (Tier 1-2 > Tier 3 > Tier 4-6 > Tier 7-10)
    3. Keep the most concise pattern description
    4. Preserve unique insights from both sources

    **For each duplicate group:**

    1. Compare semantic signatures:
       - If identical: Merge into single document
       - If similar (>80% overlap): Merge with variant section
       - If different enough: Keep separate (not actually duplicates)

    2. Merge strategy:
       ```yaml
       merged_doc:
         base: {higher authority source}
         additions:
           - from: {lower authority}
             section: {detection|fixes|examples}
             content: {unique insight}
       ```

    3. Execute merge:
       - Use higher authority doc as base
       - Add unique content from other sources
       - Ensure result < 100 lines
       - Delete redundant docs

    4. Update cross-references:
       - If doc was moved/merged, update any references
       - Update index.yaml

    **Quality checks during merge:**
    - No variable names (semantic-only)
    - BSKG operations referenced
    - Under 100 lines
    - Code examples under 15 lines

    **Document all merge decisions:**
    ```yaml
    merge_decisions:
      - group_id: 1
        action: MERGED
        base_doc: "reentrancy/classic/core-pattern.md"
        merged_from: ["reentrancy/callback/core-pattern.md"]
        reason: "Same R:bal->X:out->W:bal signature"
        additions: ["Callback-specific detection note"]
    ```
  </action>
  <verify>
    - All duplicate groups processed
    - Merged docs under 100 lines
    - No orphaned references
    - Semantic-only constraint maintained
    - Merge decisions documented
  </verify>
  <done>Pattern deduplication complete, duplicates merged with documented decisions</done>
