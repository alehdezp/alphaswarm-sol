### Task T2: Update PropulsionEngine Imports and Usage

**PropulsionConfig Changes:**
- Renamed `enable_worktree_isolation` field to `enable_workspace_isolation`

- Updated `to_dict()` to use new field name

**PropulsionEngine Changes:**
- Renamed `_worktree_manager` attribute to `_workspace_manager`
- Updated initialization to use `enable_workspace_isolation`
- Updated all internal method calls: `allocate()`, `release()`, `cleanup_pool()`
- Added new methods:
  - `cleanup_workspaces()` - replacement for `cleanup_worktrees()`
  - `get_workspace_manager()` - replacement for `get_worktree_manager()`
- Added deprecated aliases with warnings:
  - `cleanup_worktrees()` - warns and calls `cleanup_workspaces()`
  - `get_worktree_manager()` - warns and returns `_workspace_manager`

## Migration Guide for Users

### Before (deprecated)
```python
config = PropulsionConfig(
    enable_worktree_isolation=True,  # DEPRECATED
    pool_id="audit-001",
)
engine = PropulsionEngine(runtime, inboxes, config)
engine.cleanup_worktrees()  # DEPRECATED
```

### After (recommended)
```python
config = PropulsionConfig(
    enable_workspace_isolation=True,  # NEW
    pool_id="audit-001",
)
engine = PropulsionEngine(runtime, inboxes, config)
engine.cleanup_workspaces()  # NEW
```
