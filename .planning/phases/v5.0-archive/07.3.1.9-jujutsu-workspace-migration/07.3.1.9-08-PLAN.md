## Tasks

### T1: Create Find-Replace Script

Create a script to safely batch-update the guardrail text:

```bash
#!/bin/bash
# scripts/migrate_guardrail_text.sh

OLD_TEXT='- Worktree isolation: run any commands or experiments that mutate state in a fresh git worktree;
  do not use the main worktree and do not reuse worktrees for reruns.'

NEW_TEXT='- Workspace isolation: run any commands or experiments that mutate state in a fresh jj workspace;
  do not use the main workspace and do not reuse workspaces for reruns.'

# Find all PLAN.md files
find .planning/phases -name "*-PLAN.md" -type f | while read -r file; do
    if grep -q "Worktree isolation" "$file"; then
        echo "Updating: $file"
        # Use sed with backup
        sed -i.bak "s/Worktree isolation: run any commands or experiments that mutate state in a fresh git worktree/Workspace isolation: run any commands or experiments that mutate state in a fresh jj workspace/g" "$file"
        sed -i "s/do not use the main worktree and do not reuse worktrees for reruns/do not use the main workspace and do not reuse workspaces for reruns/g" "$file"
        # Remove backup
        rm -f "$file.bak"
    fi
done
```

### T2: Execute Batch Update

Run the script:
