### T3: Update `_find_repo_root()` Method

**Current (lines 86-95):**
```python
def _find_repo_root(self) -> Path:
    """Find the root of the git repository."""
    result = subprocess.run(
        ["git", "rev-parse", "--show-toplevel"],
        capture_output=True,
        text=True,
    )
    if result.returncode != 0:
        raise WorktreeError("Not in a git repository")
    return Path(result.stdout.strip())
```

**New:**
```python
def _find_repo_root(self) -> Path:
    """Find the root of the Jujutsu repository."""
    result = subprocess.run(
        ["jj", "root"],
        capture_output=True,
        text=True,
    )
    if result.returncode != 0:
        raise WorkspaceError("Not in a jj repository")
    return Path(result.stdout.strip())
```
