## Technical Details

### Jujutsu Command Mappings

| Git Worktree | Jujutsu Workspace |
|--------------|-------------------|
| `git worktree add -b <branch> <path> <ref>` | `jj workspace add <path> --name <name> -r <ref>` |
| `git worktree remove --force <path>` | `jj workspace forget <name>` + `rm -rf <path>` |
| `git rev-parse --show-toplevel` | `jj root` |
| `git rev-parse HEAD` | `jj log -r @ --no-graph -T commit_id` |
| `git worktree prune` | No-op (jj handles automatically) |

### Backward Compatibility Aliases

```python
# In workspace.py
WorktreeMetadata = WorkspaceMetadata
WorktreeError = WorkspaceError
WorktreeManager = WorkspaceManager
DEFAULT_WORKTREE_ROOT = DEFAULT_WORKSPACE_ROOT
```
