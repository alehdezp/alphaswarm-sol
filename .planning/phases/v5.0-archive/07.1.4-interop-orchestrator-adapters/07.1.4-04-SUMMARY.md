## Decisions Made

1. **Team coordination pattern:** AutoGen uses "swarm" team type (agent-driven handoffs) as default instead of round-robin for flexible workflow
2. **Process type:** CrewAI uses "sequential" process as default for predictable task ordering in investigation workflow
3. **Evidence preservation:** AutoGen stores evidence in chat context messages, CrewAI stores in shared crew memory
4. **Code execution safety:** Disabled code execution in both adapters by default for security
5. **Human input mode:** Set to "NEVER" for AutoGen to support full automation
6. **Role backstories:** Created detailed backstories for CrewAI agents to guide investigation behavior
