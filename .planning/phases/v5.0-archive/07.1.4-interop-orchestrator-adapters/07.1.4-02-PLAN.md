  <files>src/alphaswarm_sol/adapters/agents_sdk.py</files>
  <action>
Create the OpenAI Agents SDK adapter that wraps the existing OpenAIAgentsRuntime with orchestrator semantics.

1. Define `AgentsSdkConfig(AdapterConfig)`:
   - `api_key`: Optional OpenAI API key
   - `guardrails_enabled`: bool (default True)
   - `guardrail_policy_path`: Path to skill_tool_policies.yaml
   - `tracing_enabled`: bool (default True)
   - `max_turns`: int (default 10)

2. Implement `AgentsSdkAdapter(OrchestratorAdapter)`:
   - `__init__(config: AgentsSdkConfig)`:
     - Initialize OpenAIAgentsRuntime internally
     - Load guardrail policies from YAML
     - Set up trace context storage

   - `async execute_agent(config: AgentConfig, messages: List[Dict]) -> AgentResponse`:
     - Apply input guardrails from policy (check role against allowed tools)
     - Delegate to internal OpenAIAgentsRuntime.execute()
     - Apply output guardrails (validate response format)
     - Return AgentResponse with trace metadata

   - `async handoff(ctx: HandoffContext) -> HandoffResult`:
     - Create new Agent for target_agent role
     - Transfer evidence snapshot via context
     - Preserve trace_id and create child span_id
     - Execute target agent with transferred context
     - Return HandoffResult with evidence_preserved=True if bead unchanged

   - `async spawn_with_trace(config: AgentConfig, task: str, trace: TraceContext) -> AgentResponse`:
     - Create span under trace context
     - Execute agent
     - Record span events (start, tool_calls, finish)
     - Return response with trace metadata

   - `get_capabilities() -> Set[AdapterCapability]`:
     - Return capabilities from ADAPTER_CAPABILITIES["agents-sdk"]

   - `export_trace(trace: TraceContext) -> Dict[str, Any]`:
     - Export trace in OpenTelemetry-compatible format

   - `import_trace(data: Dict[str, Any]) -> TraceContext`:
     - Import trace from OpenTelemetry or custom format

   - `preserve_evidence(bead: VulnerabilityBead, ctx: HandoffContext) -> VulnerabilityBead`:
     - Validate bead fields unchanged (context_hash match)
     - Update work_state with handoff info
     - Return bead with updated metadata

3. Add private helper methods:
   - `_load_guardrail_policies(path: Path) -> Dict[str, Any]`
   - `_apply_input_guardrails(role: str, tools: List[Dict]) -> List[Dict]` (filter tools by policy)
   - `_validate_output_guardrails(role: str, response: AgentResponse) -> bool`

Import from agents SDK: Agent, Runner, FunctionTool, InputGuardrail, OutputGuardrail (if available).
