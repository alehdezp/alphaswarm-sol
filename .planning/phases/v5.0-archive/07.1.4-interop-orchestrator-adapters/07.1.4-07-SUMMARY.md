## What Changed

### LangGraphAdapter (langgraph.py)

**Before:** Referenced non-existent `AgentExecutor` from `alphaswarm_sol.agents.runtime.executor`
```python
from alphaswarm_sol.agents.runtime.executor import AgentExecutor
executor = AgentExecutor()
return await executor.execute(config, messages)
```

**After:** Uses runtime factory with OpenCode default
```python
from alphaswarm_sol.agents.runtime.factory import create_runtime, RuntimeType
self._runtime = create_runtime(sdk=RuntimeType.OPENCODE)
response = await self._runtime.execute(config, messages)
response.metadata["adapter"] = "langgraph"
```

### ClaudeCodeAdapter (claude_code.py)

**Before:** Had placeholder comment in handoff method
```python
# Execute target agent
# (placeholder - real implementation would execute claude CLI)
```

**After:** Delegates to ClaudeCodeRuntime
```python
from alphaswarm_sol.agents.runtime.claude_code import ClaudeCodeRuntime
self._runtime = ClaudeCodeRuntime(config=runtime_config)
response = await self._runtime.execute(config, messages)
```
