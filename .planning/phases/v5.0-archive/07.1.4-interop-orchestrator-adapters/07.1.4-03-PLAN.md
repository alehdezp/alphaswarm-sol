  <name>Task 1: Implement LangGraphAdapter</name>
  <files>src/alphaswarm_sol/adapters/langgraph.py</files>
  <action>
Create the LangGraph adapter that executes VRS agents as state graph nodes.

1. Define `LangGraphConfig(AdapterConfig)`:
   - `persistence_path`: Path for checkpoint storage (default ".langgraph_checkpoints")
   - `enable_replay`: bool (default True)
   - `max_concurrent_nodes`: int (default 4)
   - `state_schema`: Optional custom state schema

2. Define `VrsState` TypedDict for graph state:
   ```python
   class VrsState(TypedDict):
       bead: Dict[str, Any]  # Serialized VulnerabilityBead
       messages: List[Dict[str, Any]]  # Conversation history
       current_agent: str  # Current agent role
       evidence: Dict[str, Any]  # Accumulated evidence
       trace_context: Dict[str, Any]  # Trace propagation
       checkpoint_id: str  # For replay
   ```

3. Implement `VrsStateGraph` class:
   - `__init__(adapter: LangGraphAdapter)`:
     - Create StateGraph with VrsState
     - Add nodes for each VRS role (attacker, defender, verifier)
     - Add conditional edges based on investigation flow

   - `_create_agent_node(role: AgentRole) -> Callable`:
     - Return async function that executes agent and updates state
     - Preserve evidence in state after each node

   - `_route_next_agent(state: VrsState) -> str`:
     - Determine next agent based on current state
     - Support attacker -> defender -> verifier flow
     - Return END if verdict reached

4. Implement `LangGraphAdapter(OrchestratorAdapter)`:
   - `__init__(config: LangGraphConfig)`:
     - Initialize BeadCheckpointer
     - Create VrsStateGraph
     - Compile graph with checkpointer

   - `async execute_agent(config: AgentConfig, messages: List[Dict]) -> AgentResponse`:
     - Convert to graph input state
     - Execute single node
     - Return AgentResponse from node output

   - `async handoff(ctx: HandoffContext) -> HandoffResult`:
     - Update graph state with handoff context
     - Trigger next node execution
     - Checkpoint after handoff
     - Return HandoffResult with checkpoint_id

   - `async execute_workflow(bead: VulnerabilityBead, config: Dict) -> VulnerabilityBead`:
     - Execute full graph until END
     - Return updated bead with verdict

   - `async replay_from_checkpoint(checkpoint_id: str) -> VrsState`:
     - Load state from checkpointer
     - Resume graph execution from checkpoint

   - `get_capabilities() -> Set[AdapterCapability]`:
     - Return capabilities from ADAPTER_CAPABILITIES["langgraph"]

   - `preserve_evidence(bead: VulnerabilityBead, ctx: HandoffContext) -> VulnerabilityBead`:
     - Update bead.work_state with graph state snapshot
     - Preserve deterministic history

Use lazy imports for langgraph to avoid hard dependency: `from langgraph.graph import StateGraph, END`

<task type="auto">
  <name>Task 2: Implement BeadCheckpointer</name>
  <files>src/alphaswarm_sol/adapters/checkpointer.py</files>
  <action>
Create a bead-aware checkpointer that enables deterministic replay.

1. Define `CheckpointState` dataclass:
   - `checkpoint_id`: Unique ID (UUID)
   - `bead_id`: Associated VulnerabilityBead ID
   - `graph_state`: Serialized VrsState
   - `timestamp`: ISO timestamp
   - `node_executed`: Last node that was executed
   - `parent_checkpoint_id`: For replay chain

2. Implement `BeadCheckpointer`:
   - `__init__(storage_path: Path, bead_storage: Optional[BeadStorage] = None)`:
     - Create storage directory
     - Initialize bead storage for state persistence

   - `save(state: VrsState, node: str) -> str`:
     - Generate checkpoint_id
     - Serialize state to JSON
     - Store alongside bead state
     - Return checkpoint_id

   - `load(checkpoint_id: str) -> CheckpointState`:
     - Load checkpoint from storage
     - Validate bead still exists
     - Return CheckpointState

   - `list_checkpoints(bead_id: str) -> List[CheckpointState]`:
     - Return all checkpoints for a bead
     - Ordered by timestamp

   - `get_replay_chain(checkpoint_id: str) -> List[CheckpointState]`:
     - Build chain from initial checkpoint to given checkpoint
     - Used for deterministic replay

   - `cleanup_old_checkpoints(max_age_hours: int = 24) -> int`:
     - Remove checkpoints older than max_age
     - Return count removed

3. Implement LangGraph Checkpointer protocol (if using langgraph.checkpoint):
   - `get(config: RunnableConfig) -> Optional[CheckpointTuple]`
   - `put(config: RunnableConfig, checkpoint: Checkpoint) -> RunnableConfig`
   - `list(config: RunnableConfig) -> Iterator[CheckpointTuple]`

4. Add checkpoint validation:
   - `validate_checkpoint(state: CheckpointState) -> bool`
   - Verify bead hash matches stored hash
   - Verify state schema is valid

Storage format: JSONL files per bead_id in storage_path directory.
