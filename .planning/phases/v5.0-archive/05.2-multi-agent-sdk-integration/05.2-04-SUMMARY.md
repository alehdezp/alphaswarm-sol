## Key Implementations

### IntegratorAgent

Core integration workflow:
1. Collect verdicts from attacker, defender, verifier
2. Detect conflicts (attacker vulnerable + defender safe)
3. If conflict + auto_debate: route to DebateOrchestrator
4. If no conflict: merge evidence with content-hash deduplication
5. Determine vulnerability based on verifier > confidence priority
6. Return MergedVerdict with rationale

### Verdict Precedence

1. **Verifier** verdict takes precedence when present
2. If attacker + defender agree, use that verdict
3. If they disagree, higher confidence wins
4. Single agent: use available verdict
