## Key Implementations

### agent-run CLI Command

```bash
alphaswarm orchestrate agent-run ./contracts/ \
  --sdk anthropic \
  --attackers 2 \
  --defenders 2 \
  --verifiers 1 \
  --timeout 3600 \
  --headless \
  --verbose
```

Options:
- `--sdk`: SDK selection (anthropic/openai)
- `--attackers/defenders/verifiers`: Agent counts per role
- `--timeout`: Maximum execution time
- `--resume`: Resume from existing pool checkpoint
- `--headless`: CI-friendly mode without interactive prompts
- `--output`: Output directory for artifacts

### get_runtime() Function

```python
def get_runtime(sdk: str, config=None):
    """SDK selection: anthropic or openai"""
    if sdk == "anthropic":
        return AnthropicRuntime(config)
    elif sdk == "openai":
        return OpenAIAgentsRuntime(config)
    else:
        raise ValueError(f"Unknown SDK: {sdk}")
```
