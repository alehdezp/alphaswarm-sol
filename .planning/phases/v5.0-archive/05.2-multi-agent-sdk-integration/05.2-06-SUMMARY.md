## Key Implementations

### ConfidenceElevator

Confidence elevation based on test results:
- **Pass** = CONFIRMED (vulnerability verified)
- **Compile fail** = Inconclusive (no change)
- **Revert** = UNCERTAIN (possible mitigation)

```python
elevator = ConfidenceElevator()
result = elevator.elevate_on_test(bead, VerdictConfidence.LIKELY, test_result)
if result.elevated:
    bead = elevator.apply_elevation(bead, result)
    # bead.human_flag = False, work_state["tests_run"] populated
```

### PoCNarrativeGenerator

Two modes:

## IntegratorAgent Wiring Documentation

Per `must_haves.key_links`, IntegratorAgent calls ConfidenceElevator after test execution:

```python
# In IntegratorAgent after TestBuilderAgent completes:
from true_vkg.agents.confidence import ConfidenceElevator

elevator = ConfidenceElevator()
result = elevator.elevate_on_test(
    bead,
    current_confidence,
    test_result,  # From TestBuilderAgent.generate_test()
)
if result.elevated:
    bead = elevator.apply_elevation(bead, result)
```

Flow:
1. TestBuilderAgent generates and runs test -> GeneratedTest
2. IntegratorAgent receives test result
3. IntegratorAgent instantiates ConfidenceElevator
4. IntegratorAgent calls `elevator.elevate_on_test()` to compute result
5. IntegratorAgent calls `elevator.apply_elevation()` to update bead
