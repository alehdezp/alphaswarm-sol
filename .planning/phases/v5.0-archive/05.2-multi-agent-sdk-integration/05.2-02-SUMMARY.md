## Decisions Made

1. **Priority ordering per PHILOSOPHY.md:** Implemented 7-level priority system where critical+exploitable is highest (1), low severity is lowest (7). Exploitability detected from pattern_context.why_flagged or metadata.
2. **AgentRole in hooks module:** Defined AgentRole enum in hooks/__init__.py to avoid circular imports with agents/runtime.py which doesn't exist.
3. **Lazy heap removal:** Instead of O(n) removal from heap, we track valid bead IDs in a set and skip stale entries during pop/peek.
4. **FIFO within priority:** Beads at same priority are processed in order added (timestamp tiebreaker) for fairness.
