## What Changed

### Task 1: E2E Test Fixtures with DeterministicRuntime

Created `tests/e2e/` package with shared test infrastructure:

**DeterministicRuntime** (`fixtures.py`):
- Implements `AgentRuntime` abstract base class
- Returns predefined responses based on agent role
- Maintains call log for replay verification
- Tracks usage statistics for cost validation
- Key property: `same inputs -> same outputs`

**Deterministic Response Templates**:
```python
DETERMINISTIC_RESPONSES = {
    AgentRole.ATTACKER: "## Analysis\nThe vulnerability is exploitable...",
    AgentRole.DEFENDER: "## Analysis\nChecking for mitigations...",
    AgentRole.VERIFIER: "## Cross-Check\nVerifying claims...",
    AgentRole.TEST_BUILDER: "## Test Generation\nGenerated Foundry test...",
    AgentRole.SUPERVISOR: "## Supervisor Report\nQueue Status...",
    AgentRole.INTEGRATOR: "## Integration Report\nMerging verdicts...",
}
```

**Factory Helpers**:
- `create_minimal_bead()`: Complete bead with all required fields
- `vulnerable_contract_dir`: Temp directory with reentrancy-vulnerable contract
- `sample_bead`, `sample_pool`, `multi_bead_pool`: Standard test fixtures
