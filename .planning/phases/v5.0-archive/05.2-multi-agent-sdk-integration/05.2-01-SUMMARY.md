### AgentRuntime Interface

```python
class AgentRuntime(ABC):
    async def execute(config: AgentConfig, messages: List[Dict]) -> AgentResponse
    async def spawn_agent(config: AgentConfig, task: str) -> AgentResponse
    def get_model_for_role(role: AgentRole) -> str
    def get_usage() -> Dict[str, Any]
```

### Anthropic Prompt Caching

- Cache control markers on system prompt and tools
- 90% cost reduction on cached reads per TOKEN-EFFICIENCY-RESEARCH.md
- Cache breakeven: ~3 requests with same prefix

### Error Handling (per 05.2-CONTEXT.md)

- **Retry:** Rate limits, connection errors, timeouts, server errors (5xx)
- **Fail fast:** Authentication errors, bad requests (4xx)
- Exponential backoff with configurable base (default 2.0s)

runtime = create_runtime(RuntimeConfig(
    preferred_sdk="anthropic",
    enable_prompt_caching=True,
))

response = await runtime.execute(
    AgentConfig(
        role=AgentRole.ATTACKER,
