### AgentCoordinator (src/true_vkg/agents/propulsion/coordinator.py)

Multi-agent coordination with verifier assignment logic:

```python
class AgentCoordinator:
    """Coordinates multiple agents processing a pool of beads.

    Key: Verifier assigned to queue when BOTH attacker AND defender complete.
    """

    def _on_attacker_complete(self, bead_id: str, response) -> None:
        self._attacker_complete.add(bead_id)
        self._check_verifier_assignment(bead_id)

    def _check_verifier_assignment(self, bead_id: str) -> None:
        if bead_id in self._attacker_complete and bead_id in self._defender_complete:
            # Assign to verifier queue
            self._verifier_queue.append(bead_id)
```

Key features:
- **CoordinatorConfig**: agents_per_role mapping
- **CoordinatorStatus**: IDLE, RUNNING, PAUSED, COMPLETE, FAILED
- **CoordinatorReport**: total_beads, completed, failed, by_role, duration
- **Verifier assignment gate**: Only when BOTH complete
