## Decisions Made

1. **Contradiction agent is refutation-only**: Never confirms findings; only challenges with evidence-backed counterarguments. This ensures adversarial review remains focused.

2. **Pattern composition uses algebraic operators**: Sequence (;), parallel (||), amplification (+), and gating (~>) operators enable systematic composition of vulnerability patterns.

3. **Confidence bounds instead of point estimates**: Finding synthesizer computes lower (pessimistic) and upper (optimistic) bounds for confidence, reflecting the uncertainty in synthesis.

4. **All agents use claude-sonnet-4**: For cost-efficiency in batch discovery where many agents may run in parallel.
