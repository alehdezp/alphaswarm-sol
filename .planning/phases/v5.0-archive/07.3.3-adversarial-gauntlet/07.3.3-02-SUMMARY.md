## Cross-Contract Reasoning Traps

Designed to test genuine understanding:

1. **AC-VULN-001 (Upgrade in Proxy):** Must trace proxy -> implementation delegatecall to detect missing auth
2. **OR-VULN-002 (Decimal Propagation):** Must trace oracle.decimals() return value through calculations
3. **AC-SAFE-001 (Inherited Protection):** Must resolve OwnableUpgradeable inheritance to avoid false positive
