## Implementation Details

### Task 1: Reference Extraction Tooling

Created `src/alphaswarm_sol/vulndocs/reference_extract.py` with:
- `URLEntry` dataclass for provenance entries
- `URLLedger` class for append-only ledger management
- `extract_from_vulndocs()` - scans index.yaml and research/*.md files
- `update_ledger()` - merges new entries without clobbering existing
- `generate_provenance_report()` - coverage statistics
- `find_missing_provenance()` - gap detection

Created `scripts/vulndocs_reference_extract.py` CLI with commands:
- `extract` - Extract URLs and update ledger (supports --dry-run, --category)
- `report` - Generate provenance coverage report
- `missing` - List URLs missing from ledger
- `stats` - Show ledger statistics
