### Bead Structure

- **ContextMergeBead:** Contains full context bundle inline (self-contained)
  - id (CTX-{hash})
  - vulnerability_class
  - protocol_name
  - context_bundle (embedded ContextBundle)
  - verification_score
  - finding_bead_ids (downstream tracking)

- **VulnerabilityBead:** Contains evidence chain in metadata
  - id (VKG-{hash})
  - vulnerability_class
  - severity
  - confidence
  - metadata.evidence_chain (vulndoc_reference, code_locations, reasoning_steps, vql_queries)
  - metadata.context_bead_id (upstream reference)

**Verified:** Bidirectional linking between context and finding beads.
