# Phase 07.3.1.6 Plan Index

**Purpose:** Map every plan seed to an execution plan and show the intended execution order.

## Execution Model

All plans execute **autonomously via Claude Code controller + claude-code-controller**, with machine-verifiable validation gates:
- Execution is real (LIVE), using claude-code-controller with dedicated demo sessions
- No manual operator intervention during runs (controller drives claude-code-controller)
- Self-validation via explicit gates (transcript quality, marker presence, artifact existence)
- Human confirmation only at plan completion when confidence < 0.95
- Anti-fabrication checks prevent false evidence (line count, duration, metrics variance)

See `PLAN-TEMPLATE-AUTONOMOUS.md` for the standard plan structure.
