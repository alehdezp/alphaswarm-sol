## CLI Commands Delivered

### validate
```bash
uv run alphaswarm vulndocs validate [path]
  --strict    # Treat warnings as errors
  --json      # Output JSON for CI
```

Shows progressive validation results with:
- Rich table of vulnerabilities with validation levels
- Detailed issues (errors, warnings, suggestions)
- Summary counts
- Exit code 1 if errors (or warnings with --strict)

### scaffold
```bash
uv run alphaswarm vulndocs scaffold <category> <subcategory>
  --severity <critical|high|medium|low>
  --root <path>
```

Creates new vulnerability from templates:
- Full folder structure (index.yaml, *.md files, patterns/)
- Template variable substitution
- Input validation (lowercase, hyphens)
- Rich tree display of created structure

### info
```bash
uv run alphaswarm vulndocs info [path]
```

Shows framework statistics:
- Counts: categories, vulnerabilities, patterns
- Validation level distribution
- Coverage gaps (vulnerabilities without patterns)

### list
```bash
uv run alphaswarm vulndocs list [path]
  --category <name>
  --level <minimal|standard|complete|excellent>
```

Lists vulnerabilities with filtering:
- Rich table with path, validation level, issue count
- Filter by category or validation level
- Level icons (⚪ ➔ 🟡 ➔ 🟢 ➔ 🌟)
