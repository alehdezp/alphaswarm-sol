    1. VulnDocIndex (BaseModel):
       Required fields:
       - id: str
       - category: str
       - subcategory: str
       - severity: Severity
       - vulndoc: str  # Self-reference path for validation

       Phase 7 test generation fields (optional but recommended):
       - semantic_triggers: list[str] = []
       - vql_queries: list[str] = []
       - graph_patterns: list[str] = []
       - reasoning_template: str | None = None

       Pattern and test tracking:
       - patterns: list[str] = []
       - test_coverage: list[str] = []

       Metadata:
       - description: str | None = None
       - severity_range: list[Severity] = []
       - related_exploits: list[str] = []
       - created: str | None = None
       - updated: str | None = None

       Validators:
       - validate_semantic_triggers: Check against VALID_SEMANTIC_OPERATIONS
       - validate_vulndoc_path: Must match "{category}/{subcategory}" format

    2. PatternRef (BaseModel):
       Required fields:
       - id: str (pattern ID like "oracle-001-twap")
       - name: str
       - severity: Severity
       - vulndoc: str  # CRITICAL: links to vulndoc folder

       Optional fields:
       - scope: str = "Function"
       - lens: list[str] = []
       - match: dict | None = None  # tier_a/tier_b/tier_c structure

       Test tracking:
       - test_coverage: TestCoverage | None = None

    3. TestCoverage (BaseModel):
       - precision: float | None = None
       - recall: float | None = None
       - status: str = "draft"  # draft, ready, excellent

    4. VulnDocCategory (BaseModel):
       - id: str
       - name: str
       - overview: str | None = None
       - subcategories: list[str] = []

    5. Helper functions:
       - generate_json_schema() -> dict: Generate JSON Schema from models
       - load_vulndoc_index(path: Path) -> VulnDocIndex
       - load_pattern_ref(path: Path) -> PatternRef

    Use pydantic 2.x patterns (model_validator, field_validator).
