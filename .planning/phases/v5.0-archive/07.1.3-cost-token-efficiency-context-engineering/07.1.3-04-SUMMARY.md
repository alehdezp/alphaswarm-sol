## Decisions Made

1. **In-memory cache for graph queries** - Faster than disk, queries are typically repeated within a session
2. **Deterministic cache keys** - SHA256 of combined inputs ensures reproducibility
3. **LRU eviction by access count** - Simple approximation; could use OrderedDict for true LRU
4. **Pool/run scoping in tool cache** - Enables isolated cache per orchestration pool/run
