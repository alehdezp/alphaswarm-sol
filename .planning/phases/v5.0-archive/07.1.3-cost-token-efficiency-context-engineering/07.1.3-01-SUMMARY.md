## Implementation Details

### Context Budget Policy (Task 1)

Created `src/alphaswarm_sol/llm/context_budget.py` with:

- **ContextBudgetStage**: Enum for disclosure stages (SUMMARY=15%, EVIDENCE=50%, RAW=100% of budget)
- **ContextBudgetPolicy**: Enforces budgets with deterministic trimming
- **ContextBudgetReport**: Tracks original/final tokens, dropped sections, preserved evidence IDs
- **Role budgets**: classifier=2000, attacker=6000, defender=5000, verifier=4000
- **Pool budgets**: triage=2000, investigation=6000, verification=4000
- **Token estimation**: 4-chars-per-token heuristic (no external tokenizer)

Key methods:
- `apply_budget(context, stage)` - Trims context to fit stage budget
- `expand_context(current, additional, to_stage)` - Progressive expansion
- `get_stage_budget(stage)` - Returns token budget for stage
