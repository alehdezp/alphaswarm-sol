<task type="auto">
  <name>Task 2: Test distribution modes and optimize configuration</name>
  <files>.planning/phases/08-test-performance-research/08-02-xdist-results.txt</files>
  <action>
Benchmark different distribution modes to find optimal configuration for AlphaSwarm.sol (per 08-RESEARCH.md architecture patterns):

```bash
echo "" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
echo "=== Distribution Mode Comparison ===" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt

# Mode 1: loadfile (groups tests by file - recommended for load_graph LRU cache)
echo "" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
echo "--- loadfile mode (recommended for LRU cache) ---" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
time uv run pytest tests/ -n auto --dist loadfile 2>&1 | tail -20 >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt

# Mode 2: loadscope (groups by test class/module)
echo "" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
echo "--- loadscope mode ---" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
time uv run pytest tests/ -n auto --dist loadscope 2>&1 | tail -20 >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt

# Mode 3: load (default - round robin)
echo "" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
echo "--- load mode (default round robin) ---" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
time uv run pytest tests/ -n auto --dist load 2>&1 | tail -20 >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
```

Per 08-RESEARCH.md: Given the `@lru_cache` on `load_graph()`, `--dist loadfile` should perform best as tests sharing the same contract file will run on the same worker.

Also test worker counts to find optimal:
```bash
echo "" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
echo "=== Worker Count Comparison ===" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
for workers in 2 4 8; do
  echo "" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
  echo "--- $workers workers with loadfile ---" >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
  time uv run pytest tests/ -n $workers --dist loadfile 2>&1 | tail -10 >> .planning/phases/08-test-performance-research/08-02-xdist-results.txt
done
```
