# Phase 5.9 Plan 04: Deterministic Evidence IDs + Graph Build Hash

Deterministic evidence references tied to graph build hashes for reproducible findings.

## Summary

Implemented deterministic evidence ID generation and resolution system that ties all findings
to specific graph builds. Evidence IDs use the format `EVD-<8-hex>` computed from
`SHA256(build_hash:node_id:line:column)[:8]`, ensuring reproducibility across runs.

## Key Components Created

### Evidence IDs (src/alphaswarm_sol/llm/evidence_ids.py)

- **EvidenceID**: Frozen dataclass with deterministic ID from build_hash + node_id + span
- **EvidenceRegistry**: Tracks registered evidence for bidirectional resolution
- **SourceSpan**: Source code range representation with validation
- **ClauseMatrixBuilder**: Constructs v2 contract clause_matrix with evidence linkage
- Helper functions: `generate_evidence_id`, `validate_evidence_id`, `build_evidence_ref`

### Graph Build Hash (src/alphaswarm_sol/kg/graph_hash.py)

- **compute_graph_hash**: Deterministic hash from graph content (order-independent)
- **compute_source_hash**: Hash from source file contents
- **compute_incremental_hash**: Update hash with new content
- **BuildHashTracker**: Tracks hash history across operations
- Validation: `validate_build_hash`, `check_build_hash_consistency`
- Embedding: `embed_build_hash`, `extract_build_hash`

## API Examples

```python
from alphaswarm_sol.llm.evidence_ids import (
    EvidenceID, EvidenceRegistry, SourceSpan, ClauseMatrixBuilder
)
from alphaswarm_sol.kg.graph_hash import compute_graph_hash

# Compute build hash from graph
graph = {"nodes": [...], "edges": [...]}
build_hash = compute_graph_hash(graph)  # "a1b2c3d4e5f6"

# Create evidence registry
registry = EvidenceRegistry(build_hash=build_hash)

# Register evidence
span = SourceSpan(file="Token.sol", line_start=42, column_start=4)
evidence = registry.register("N-withdraw-001", span)
print(evidence.id)  # "EVD-f3a2b1c0"

# Resolve back to source
resolved = registry.resolve("EVD-f3a2b1c0")
print(resolved.span.file)  # "Token.sol"

# Build clause matrix
builder = ClauseMatrixBuilder(registry)
builder.add_matched("reentrancy:all:0", "N-001", span)
builder.add_unknown("reentrancy:all:1", "external_return_untracked")
matrix = builder.build()
```
