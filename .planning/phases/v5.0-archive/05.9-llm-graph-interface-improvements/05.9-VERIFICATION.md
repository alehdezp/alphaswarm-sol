### Gaps Summary

Two blockers prevent the phase goal from being fully achieved:

1. **Unified slicing pipeline not wired into LLM-facing surfaces** — router and LLM workflow still use local slicers, so the single unified pipeline is not actually used. This breaks the “one slicing pipeline” guarantee and risks inconsistent omissions/coverage semantics across agents.
