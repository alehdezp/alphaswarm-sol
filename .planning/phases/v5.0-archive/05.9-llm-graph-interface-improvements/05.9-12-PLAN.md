<research_references>
**Critical Pitfall (PITFALLS.md):** "Silent Omissions Cause False Safety" - When a subgraph extraction prunes nodes or edges, the LLM receives a smaller graph and assumes it's complete. Missing nodes are interpreted as "no evidence of vulnerability" rather than "evidence not included." This leads to false safety conclusions.

**Prevention:** Mandatory omission ledger on ALL subgraph outputs. Coverage score as a first-class field. Regression tests with known omissions.

**Detection:** Coverage score < 0.8 should trigger warning. Agent asks "is this all the relevant code?" with no answer = problem.
</research_references>

   ```python
   def test_coverage_score_deterministic():
       """Same graph + same query = same coverage score."""
       for _ in range(5):
           result = extract_subgraph(graph, seed_nodes=["func_withdraw"])
           scores.append(result["omissions"]["coverage_score"])

       assert all(s == scores[0] for s in scores), \
           f"Coverage scores not deterministic: {scores}"

   def test_coverage_score_rebuild_deterministic():
       """Rebuilding graph from same source = same coverage score."""
       graph1 = build_kg(contract_path)
       graph2 = build_kg(contract_path)

       result1 = extract_subgraph(graph1, seed_nodes=["func_withdraw"])
       result2 = extract_subgraph(graph2, seed_nodes=["func_withdraw"])

       assert result1["omissions"]["coverage_score"] == \
              result2["omissions"]["coverage_score"]
   ```

2. **Build hash stability:**
   ```python
   def test_build_hash_deterministic():
       """Build hash is deterministic for same source."""
       graph1 = build_kg(contract_path)
       graph2 = build_kg(contract_path)

       assert graph1.build_hash == graph2.build_hash

   def test_evidence_ids_stable_across_builds():
       """Evidence IDs are stable when build hash matches."""
       graph1 = build_kg(contract_path)
       graph2 = build_kg(contract_path)

       findings1 = query_pattern(graph1, "reentrancy-classic")
       findings2 = query_pattern(graph2, "reentrancy-classic")

       # Extract evidence IDs
       ids1 = {e["deterministic_id"] for f in findings1 for e in f["evidence_refs"]}
       ids2 = {e["deterministic_id"] for f in findings2 for e in f["evidence_refs"]}

       assert ids1 == ids2
   ```

3. **Coverage formula validation:**
   ```python
   def test_coverage_formula_matches_spec():
       """Coverage = captured_weight / relevant_weight per spec."""
       result = extract_subgraph(graph, seed_nodes=["func_withdraw"])

       # Manually compute expected coverage
       captured = len(result["nodes"])
       relevant = len(result["omissions"].get("relevant_nodes", [])) + captured

       expected = captured / relevant if relevant > 0 else 1.0
       actual = result["omissions"]["coverage_score"]

       assert abs(actual - expected) < 0.01, \
           f"Coverage formula mismatch: expected {expected}, got {actual}"
   ```

   ```python
   def test_omission_ledger_fields_consistent():
   ```
