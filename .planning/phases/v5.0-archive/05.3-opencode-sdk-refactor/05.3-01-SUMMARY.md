## Decisions Made

1. **TaskType enum structure:** 10 values covering all task categories (VERIFY, SUMMARIZE, CONTEXT, CODE, REASONING, REASONING_HEAVY, HEAVY, ANALYZE, REVIEW, CRITICAL)

2. **Model defaults per task type:**
   - VERIFY/SUMMARIZE: `minimax/minimax-m2:free` (free tier)
   - CONTEXT: `x-ai/grok-code-fast-1` (free)
   - CODE: `zhipu/glm-4.7` (subscription)
   - REASONING: `deepseek/deepseek-v3.2` (low cost)
   - REASONING_HEAVY: `google/gemini-3-pro-preview` (quality)
   - HEAVY/ANALYZE: `google/gemini-3-flash-preview` (default)
   - Fallback: `qwen/qwen-2.5-72b-instruct:free`

3. **Loop prevention constants:**
   - MAX_ITERATIONS = 10 (hard limit per task)
   - MAX_REPEATED_OUTPUTS = 3 (consecutive identical outputs)
   - TOKEN_CEILING = 100,000 (max tokens per task)

4. **Rankings format:** Simple YAML with task-type keys mapping to model lists
