# Phase 5.3 Plan 09: CLI Updates Summary

**One-liner:** CLI runtime selection with --runtime flag and model benchmark commands under `benchmark models`.

## What Was Built

### Task 1: Update orchestrate CLI
Updated `src/true_vkg/cli/orchestrate.py` with new flags and commands:

1. **--runtime / -r flag** for agent-run command:
   - Options: opencode, claude-code, codex, api, auto
   - Default: auto (uses opencode for cost optimization)
   - Replaces deprecated --sdk flag (still works for backward compat)

2. **--show-rankings flag**: Display current model rankings before execution

3. **--reset-rankings flag**: Clear existing rankings and start fresh

4. **API usage warning**: When using api runtime, shows cost warning and requires confirmation

5. **New orchestrate subcommands**:
   - `orchestrate rankings`: Display/manage model rankings with --task-type filter
   - `orchestrate runtimes`: List available runtimes with availability status

**Key additions:**
```python
class RuntimeChoice(str, Enum):
    OPENCODE = "opencode"

    AUTO = "auto"
```

### Task 2: Create benchmark CLI
Created `src/true_vkg/cli/benchmark.py` (769 lines) with model benchmarking commands:

1. **benchmark models run**: Run benchmark suite on models
   - `--model`: Specific model to benchmark
   - `--category`: Specific category to test
   - `--all-models`: Benchmark all configured models
   - `--iterations`: Number of iterations per benchmark

2. **benchmark models rankings**: Show current model rankings
   - `--task-type`: Filter by task type
   - `--top`: Show top N models
   - `--format`: json/csv/table

3. **benchmark models compare**: Compare two models
   - Side-by-side comparison across all categories
   - Winner determination per category

4. **benchmark models integrate**: Integrate new model
   - Run benchmarks and update rankings
   - `--iterations` control

5. **benchmark models export**: Export benchmark results
   - `--format`: json/csv
   - `--output`: File path

6. **benchmark models list-categories**: List available benchmark categories

**Integration with main.py:**
```python
from true_vkg.cli.benchmark import benchmark_cli as model_benchmark_cli
benchmark_app.add_typer(model_benchmark_cli, name="models")
```
