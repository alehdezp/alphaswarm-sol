## Decisions Made

1. **EMA decay factor:** 0.95 (recent feedback contributes 5%)
   - Provides gradual adaptation while maintaining stability
   - First 5 samples use simple averaging to establish baseline

2. **Selection priority:**
   - latency_sensitive: sort by latency (lowest first)
   - accuracy_critical: filter by thresholds (0.85 success, 0.80 quality), sort by quality
   - Default: sort by cost (lowest first)

3. **Storage format:** YAML at .vrs/rankings/rankings.yaml
   - Human-readable for debugging
   - Git-friendly for version control
   - Atomic write with temp file + rename

4. **Capability filtering:** Models with insufficient context window excluded
   - Uses MODEL_CONTEXT_LIMITS from types.py
   - Profile.context_size compared against model limits
