## Implementation Details

### Task 1: Propulsion Engine Integration

Updated `src/true_vkg/agents/propulsion/engine.py` with:

1. **New imports and initialization:**
   - TaskRouter, ModelSelector, FeedbackCollector, RankingsStore
   - TaskType enum for routing decisions
   - TaskProfile for model selection criteria

2. **Full integration flow via `_execute_with_routing()`:**
   ```
   PropulsionEngine.execute_agent(role, task_type)
       -> ModelSelector.select(profile)     # Pick best model
       -> TaskRouter.route(policy)          # Get runtime
       -> runtime.execute(config, messages) # Execute
       -> FeedbackCollector.record(feedback)# Update rankings
   ```

3. **Fallback logic:**
   - Primary runtime failure triggers fallback to Claude Code
   - Fallback metadata recorded in response
   - `get_fallback_rate()` tracks fallback frequency

4. **Cost tracking via CostSummary:**
   - Aggregates costs by runtime, model, and role
   - `get_cost_summary()` returns breakdown
   - `cost_threshold_usd` config option to abort on threshold

5. **Role-to-task-type mapping:**
   - ATTACKER -> CRITICAL (deep reasoning)
   - DEFENDER -> ANALYZE (fast analysis)
   - VERIFIER -> CRITICAL (accuracy)
   - TEST_BUILDER -> CODE (generation)
