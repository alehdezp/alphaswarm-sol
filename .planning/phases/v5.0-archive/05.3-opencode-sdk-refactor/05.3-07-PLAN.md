<objective>
Implement model ranking system for continuous optimization.

Purpose: Track model performance per task type and use rankings to select the best model. Rankings update via EMA after each execution, enabling continuous learning.

Output: Ranking module with schemas, selector, feedback, and store.

**Integration Note:** The ModelSelector class created here will be used by OpenCodeRuntime (Plan 01) and integrated into PropulsionEngine (Plan 08). The flow is:
- PropulsionEngine calls TaskRouter
- TaskRouter uses ModelSelector to pick best model
- ModelSelector queries RankingsStore
- Selected model is passed to OpenCodeRuntime.execute()
</objective>

```python
@dataclass
class TaskProfile:
    """Profile defining task requirements for model selection."""
    task_type: TaskType
    complexity: str  # simple, moderate, complex
    context_size: int  # Required context window
    output_size: int  # Expected output size
    requires_tools: bool  # Needs tool use capability
    latency_sensitive: bool  # Needs fast response
    accuracy_critical: bool  # Quality matters most

@dataclass
class ModelRanking:
    """Track model performance per task type."""
    model_id: str
    task_type: str
    success_rate: float  # 0.0-1.0
    average_latency_ms: int
    average_tokens: int
    quality_score: float  # 0.0-1.0
    cost_per_task: float  # USD
    sample_count: int
    last_updated: datetime

    def to_dict(self) -> Dict[str, Any]
    @classmethod
    def from_dict(cls, data: Dict) -> "ModelRanking"

@dataclass
class TaskFeedback:
    """Feedback from task execution for ranking updates."""
    task_id: str
    model_id: str
    task_type: str
    success: bool
    latency_ms: int
    tokens_used: int
    quality_score: float  # 0.0-1.0
    cost_usd: float
    timestamp: datetime
    error_message: str | None = None

    def to_dict(self) -> Dict[str, Any]
    @classmethod
    def from_dict(cls, data: Dict) -> "TaskFeedback"
```
