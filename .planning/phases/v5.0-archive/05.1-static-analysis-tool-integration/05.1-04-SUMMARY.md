2. **Tool grouping strategy**:

3. **Cache key structure**: `{tool}_{project_hash[:12]}_{config_hash[:8]}.json` - Includes both project state and config to handle configuration changes.

4. **TTL-based expiration**: Default 24 hours, configurable. Automatic cleanup via `clear_expired()`.
