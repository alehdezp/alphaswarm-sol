### Task 2: Enhanced Deduplication with Semantic Similarity
**File:** `src/true_vkg/orchestration/dedup.py` (865 lines)

Enhanced existing deduplication with:
- **SemanticDeduplicator** class with two-stage deduplication:
  - Stage 1: Location-based clustering (file + line tolerance)
  - Stage 2: Semantic similarity via sentence-transformers embeddings
- **DeduplicationStats** dataclass tracking reduction metrics
- Lazy-loaded embedding model (`all-MiniLM-L6-v2`)
- Graceful fallback when sentence-transformers not installed
- Confidence boosting on tool agreement:
  - 2 tools: +0.1
  - 3+ tools: +0.2

### Task 3: Pattern Skip Logic in Coordinator
**File:** `src/true_vkg/tools/coordinator.py` (updated)

Added TOOL-07 pattern skip logic:
- **SKIP_THRESHOLD = 0.80** - Skip patterns if tool precision >= 80%
- **NEVER_SKIP_PATTERNS** - 13 patterns that tools can't detect:
  - business-logic-violation, economic-manipulation, governance-attack
  - cross-function-reentrancy, cross-contract-reentrancy
  - oracle-manipulation, price-manipulation, slippage-manipulation
  - sandwich-attack, flash-loan-attack, front-running
  - privilege-escalation, role-confusion
- `_calculate_pattern_skips()` method determines patterns to skip
- `get_edge_case_patterns()` returns protected patterns
- `explain_strategy()` includes skip rationale
