<task type="auto">
  <name>Task 1: SARIF Adapter (Universal Normalization)</name>
  <files>src/true_vkg/tools/adapters/__init__.py, src/true_vkg/tools/adapters/sarif.py</files>
  <action>
Create SARIF 2.1.0 adapter as the universal normalization layer.

**VKGFinding dataclass (internal format):**
- id: str (unique finding ID)
- source: str (tool name)
- rule_id: str (detector/rule ID)
- title: str
- description: str
- severity: str (critical/high/medium/low/info)
- category: str (reentrancy, access_control, etc.)
- file: str (relative path)
- line: int
- end_line: Optional[int]
- column: Optional[int]
- function: Optional[str]
- contract: Optional[str]
- confidence: float (0.0-1.0)
- tool_confidence: str (original tool confidence)
- raw: Dict[str, Any] (original data)

**SARIFAdapter class:**
- SARIF_VERSION = "2.1.0"
- SARIF_SCHEMA = "https://raw.githubusercontent.com/oasis-tcs/sarif-spec/master/Schemata/sarif-schema-2.1.0.json"

Methods:
- to_sarif(findings: List[VKGFinding], tool_name: str, tool_version: str) -> Dict[str, Any]
- from_sarif(sarif: Dict[str, Any]) -> List[VKGFinding]
- validate_sarif(sarif: Dict[str, Any]) -> bool (basic schema validation)
- sarif_level_to_severity(level: str) -> str
- severity_to_sarif_level(severity: str) -> str

**SARIF structure:**
```json
{
  "$schema": "...",
  "version": "2.1.0",
  "runs": [{
    "tool": {
      "driver": {
        "name": "...",
        "version": "...",
        "rules": [...]
      }
    },
    "results": [{
      "ruleId": "...",
      "level": "warning|error|note",
      "message": {"text": "..."},
      "locations": [{
        "physicalLocation": {
          "artifactLocation": {"uri": "..."},
          "region": {"startLine": ..., "endLine": ...}
        }
      }]
    }]
  }]
}
```

**Severity mapping:**
- SARIF error -> critical/high
- SARIF warning -> medium
- SARIF note -> low/info

Include module-level functions: sarif_to_vkg_findings(), vkg_findings_to_sarif() for convenience.
  </action>
  <verify>
```bash
python -c "from true_vkg.tools.adapters.sarif import SARIFAdapter, VKGFinding; print('SARIF adapter loaded')"
```
  </verify>
  <done>SARIF adapter converts between SARIF 2.1.0 and BSKG internal format bidirectionally.</done>
</task>
