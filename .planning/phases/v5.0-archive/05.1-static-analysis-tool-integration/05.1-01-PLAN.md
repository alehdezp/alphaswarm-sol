<task type="auto">
  <name>Task 1: Tool Registry Module</name>
  <files>src/true_vkg/tools/registry.py</files>
  <action>
Create tool registry with discovery, health checks, and metadata.

**ToolInfo dataclass:**
- name: str (slither, aderyn, mythril, etc.)
- tier: int (0=core, 1=recommended, 2=optional)
- binary: str (executable name)
- install_method: str (pip, cargo, curl, etc.)
- install_hint: str (installation command)
- version_cmd: List[str] (command to get version)
- health_cmd: List[str] (command to verify working)
- model_tier: str (haiku-4.5 for running, sonnet-4.5 for coordination)

**ToolHealth dataclass:**
- tool: str
- installed: bool
- version: Optional[str]
- healthy: bool
- error: Optional[str]
- last_checked: datetime

**ToolRegistry class:**
- TOOL_DEFINITIONS: Dict[str, ToolInfo] for all Tier 0/1/2 tools
- check_tool(name: str) -> ToolHealth
- check_all_tools() -> Dict[str, ToolHealth]
- get_available_tools() -> List[str]
- get_missing_tools() -> List[str]
- get_install_hint(name: str) -> str
- validate_all_before_analysis() -> bool (logs warnings for missing)

**Tools to include:**
- Tier 0: slither (pip install slither-analyzer)
- Tier 1: aderyn (cargo install aderyn), mythril (pip install mythril), echidna (binary), foundry (curl script), semgrep (pip install semgrep)
- Tier 2: halmos (uv tool install halmos)

Use shutil.which() for binary discovery. Parse version from command output.
Log warnings via structlog, never raise for missing tools.
  </action>
  <verify>
```bash
python -c "from true_vkg.tools.registry import ToolRegistry, check_all_tools; r = ToolRegistry(); print(r.get_available_tools())"
```
  </verify>

Create configuration management for tool-specific settings.

**ToolConfig dataclass:**
- tool: str
- enabled: bool = True
- timeout: int (default per tool)
- exclude_paths: List[str] (lib/, node_modules/, test/)
- exclude_detectors: List[str] (tool-specific)
- extra_args: List[str]
- custom: Dict[str, Any] (tool-specific options)

**Config loading:**
- load_tool_config(tool: str, project_path: Path) -> ToolConfig
- get_optimal_config(tool: str) -> ToolConfig (VKG defaults)
- merge_configs(base: ToolConfig, override: ToolConfig) -> ToolConfig
- save_project_config(configs: Dict[str, ToolConfig], path: Path)

**Priority order:**
1. Project-level: .vrs/tools.yaml
2. BSKG defaults: Built-in optimal configs
3. Tool defaults: What tool uses if no config

  <name>Task 3: Tool Installation Script</name>

  <action>
Create selective installation script for all supported tools.

**Script features:**
- Interactive mode: asks which tools to install
- Batch mode: --all, --tier-0, --tier-1, --tool=name
- Detects existing installations (skip if present)
- Validates after installation
- Works on macOS and Linux
- Uses best installation method per tool

**Installation methods:**
- slither: `pip install slither-analyzer`
- aderyn: `cargo install aderyn` (requires Rust)
- mythril: `pip install mythril`
- echidna: Binary from GitHub releases or `nix-env -i echidna`
- foundry: `curl -L https://foundry.paradigm.xyz | bash && foundryup`
- semgrep: `pip install semgrep`
- halmos: `uv tool install halmos` or `pip install halmos`

**Script structure:**

