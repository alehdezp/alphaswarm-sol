## Decisions Made

2. **Health cache with 5 minute TTL** - Avoid repeated subprocess calls during single session
3. **VKG-optimized defaults** - Exclude noisy detectors (naming-convention, solc-version, pragma, etc.) to focus on security-relevant findings
4. **Config cascade pattern** - Project config wins over BSKG defaults, enables customization without forking
5. **Multi-method installation** - Different tools need different install methods (pip, cargo, curl, go, nix)
