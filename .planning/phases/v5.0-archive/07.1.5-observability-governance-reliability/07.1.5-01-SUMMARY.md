## Decisions Made
- **GenAI semantic conventions v1.37+:** Use gen_ai.system (not deprecated gen_ai.provider), gen_ai.request.model, gen_ai.operation.name
- **BatchSpanProcessor:** More efficient than SimpleSpanProcessor for production use
- **Optional OTLP endpoint:** Console exporter for development, OTLP for production
- **Context managers for spans:** Automatic status/exception handling and cleanup
- **Custom alphaswarm.* attributes:** Pool/bead context propagation (alphaswarm.pool.id, alphaswarm.bead.id)
- **Event-based metadata logging:** Record token counts and metadata without full prompt payloads to avoid PII and token bloat
