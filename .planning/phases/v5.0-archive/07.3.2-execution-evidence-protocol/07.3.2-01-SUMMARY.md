## Nonce Challenge Protocol

The nonce challenge ensures proof tokens come from real claude-code-agent-teams executions:

```
1. Controller generates nonce: nonce-8f3d
2. Controller injects into claude-code-agent-teams: echo "[ALPHASWARM-NONCE] nonce-8f3d"
3. Subject echoes nonce in output (visible in transcript)
4. Proof token collector verifies nonce in transcript
5. Missing nonce -> nonce_verified=False -> INVALID proof
```

**Anti-Fake Hash Chain:**
```
nonce -> transcript -> transcript_hash -> proof_token -> evidence_pack_manifest
```

If any link is missing or mismatched, the run is invalid.
