## Decisions Made
- **Separated allow-lists:** `allowed_file_reads` uses substring match on the file path parameter; `allowed_bash_commands` uses `startswith` (prefix match) on the command string. This is the core security mechanism that closes Gap 2.7.
- **Backward compatibility:** When config has `allowed_reads` (old format) but not `allowed_file_reads`, the code falls back to `allowed_reads` for file-based tools, preserving dev-config behavior.
- **Fail-fast semantics:** `_GUARD_PROFILE=strict` + missing/invalid `DELEGATE_GUARD_CONFIG` causes immediate `sys.exit(2)`, preventing silent fallthrough to the permissive dev config.
- **10 tests instead of 9:** Added a supplementary test for blocked-call logging verification beyond the 9 specified. All tests call `delegate_guard.main()` directly (no subprocess) for speed.
