## Dockerfile (Production-Quality)

Create `/Volumes/ex_ssd/home/projects/python/vkg-solidity/alphaswarm/Dockerfile`:

```dockerfile
# Stage 1: Build environment
FROM python:3.11-slim AS builder

# Install build dependencies
RUN apt-get update && apt-get install -y --no-install-recommends \
    gcc \
    git \
    && rm -rf /var/lib/apt/lists/*

# Create virtual environment
RUN python -m venv /opt/venv
ENV PATH="/opt/venv/bin:$PATH"

# Install BSKG from PyPI (after 16.4 is complete)
# During development, use: COPY . /app && pip install /app
RUN pip install --no-cache-dir alphaswarm==4.0.0

# Stage 2: Runtime environment
FROM python:3.11-slim AS runtime

# Security: Run as non-root user
RUN useradd --create-home --shell /bin/bash vkg
USER vkg

# Copy virtual environment from builder
COPY --from=builder --chown=vkg:vkg /opt/venv /opt/venv
ENV PATH="/opt/venv/bin:$PATH"

# Set working directory (user's project will be mounted here)
WORKDIR /project

# Health check
HEALTHCHECK --interval=30s --timeout=10s --start-period=5s --retries=3 \
    CMD vkg --version || exit 1

# Default command
ENTRYPOINT ["vkg"]
CMD ["--help"]

# Labels for metadata
LABEL org.opencontainers.image.title="AlphaSwarm.sol"
LABEL org.opencontainers.image.description="Vulnerability Knowledge Graph for Solidity"
LABEL org.opencontainers.image.version="4.0.0"
LABEL org.opencontainers.image.source="https://github.com/[org]/alphaswarm"
LABEL org.opencontainers.image.licenses="MIT"
```

## Build & Test Locally

```bash
# Build image
docker build -t truevkg/alphaswarm:4.0.0 .

# Test version
docker run --rm truevkg/alphaswarm:4.0.0 --version
# Expected: BSKG 4.0.0

# Test help
docker run --rm truevkg/alphaswarm:4.0.0 --help

# Test on a real project
cd /path/to/solidity-project
docker run --rm -v $(pwd):/project truevkg/alphaswarm:4.0.0 build contracts/
docker run --rm -v $(pwd):/project truevkg/alphaswarm:4.0.0 analyze
docker run --rm -v $(pwd):/project truevkg/alphaswarm:4.0.0 findings list

# Check image size
docker images truevkg/alphaswarm:4.0.0
# Target: < 500MB
```

## Multi-Platform Build (Optional but Recommended)

For arm64 (M1/M2 Macs) and amd64 support:

```bash
# Create builder
docker buildx create --name vkg-builder --use

# Build for multiple platforms
docker buildx build \
    --platform linux/amd64,linux/arm64 \
    -t truevkg/alphaswarm:4.0.0 \
    -t truevkg/alphaswarm:latest \
    --push \
    .
```

## Troubleshooting

| Problem | Cause | Solution |
|---------|-------|----------|
| Build fails | Missing dependency | Add to apt-get install |
| "Permission denied" | Non-root user issue | Check volume permissions |
| Slither fails | solc not found | Image inherits from slither |
| Image too large | Unoptimized | Use multi-stage build |
| Push fails | Auth issue | Re-login with token |

## CI/CD Integration

Add to `.github/workflows/docker.yml`:

```yaml
name: Docker

on:
  release:
    types: [published]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Set up Docker Buildx
        uses: docker/setup-buildx-action@v3

      - name: Login to Docker Hub
        uses: docker/login-action@v3
        with:
          username: ${{ secrets.DOCKERHUB_USERNAME }}
          password: ${{ secrets.DOCKERHUB_TOKEN }}

      - name: Build and push
        uses: docker/build-push-action@v5
        with:
          context: .
          platforms: linux/amd64,linux/arm64
          push: true
          tags: |
            truevkg/alphaswarm:${{ github.event.release.tag_name }}
            truevkg/alphaswarm:latest
```
