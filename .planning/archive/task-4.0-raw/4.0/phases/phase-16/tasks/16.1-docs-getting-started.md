### 1. Prerequisites (30 seconds to verify)

```markdown
## Prerequisites

Before starting, ensure you have:

- [ ] Python 3.9 or higher: `python --version`
- [ ] pip or uv package manager
- [ ] A Solidity project with compilable contracts

Optional:
- [ ] Foundry or Hardhat (for better compilation support)
```

### 2. Installation (30 seconds)

```markdown
## Installation

```bash
# Using pip
pip install alphaswarm

# Using uv (faster)
uv pip install alphaswarm

# Verify installation
vkg --version
```

**Expected output:**
```
VKG 4.0.0
```
```

### 3. First Analysis (5 minutes)

```markdown
## Your First Analysis

### Step 1: Navigate to your project
```bash
cd /path/to/your/solidity-project
```

### Step 2: Initialize VKG
```bash
vkg init
```

This creates a `.vrs/` directory with configuration.

### Step 3: Build the knowledge graph
```bash
vkg build contracts/
```

**What you'll see:**
```
Building knowledge graph...
Found 5 contracts, 23 functions
Graph built in 2.3s
Saved to .vrs/graph.json
```

### Step 4: Run analysis
```bash
vkg analyze
```

### Step 5: View findings
```bash
vkg findings list
```

**Example output:**
```
VKG-001 [HIGH] Reentrancy in withdraw() at Vault.sol:45
VKG-002 [MEDIUM] Missing access control on setFee() at Config.sol:12
```
```

### 4. Understanding Results

```markdown
## Understanding Your Results

Each finding includes:
- **ID**: Unique identifier (e.g., VKG-001)
- **Severity**: CRITICAL, HIGH, MEDIUM, LOW, INFO
- **Location**: File and line number
- **Pattern**: Which detection pattern matched

### Get details on a specific finding:
```bash
vkg findings show VKG-001
```
```

### 5. Next Steps

```markdown
## Next Steps

- **Investigate a finding**: `vkg investigate VKG-001`
- **Generate test scaffold**: `vkg scaffold VKG-001`
- **Export for CI/CD**: `vkg report --format sarif > results.sarif`
- **Learn pattern authoring**: [Pattern Guide](guides/patterns.md)
```

### 6. Troubleshooting

```markdown
## Troubleshooting

### "Slither not found"
```bash
pip install slither-analyzer
```

### "No contracts found"
Ensure your contracts compile. Try:
```bash
forge build  # If using Foundry
npx hardhat compile  # If using Hardhat
```

### "Graph build failed"
Run with verbose output:
```bash
vkg build contracts/ --verbose
```

### Still stuck?
- [Open an issue](https://github.com/[org]/alphaswarm/issues)
- Check [known limitations](limitations.md)
```

## Validation Protocol

### Test on Fresh System

**Required: Test on at least ONE fresh environment**

Option A: Docker (fastest)
```bash
docker run -it --rm -v $(pwd):/project python:3.11-slim bash
# Inside container:
pip install alphaswarm
cd /project
# Follow getting started guide
```

Option B: GitHub Codespace
1. Create new codespace
2. Follow guide exactly
3. Note any friction

Option C: Fresh VM (most thorough)
```bash
# Create Ubuntu VM
multipass launch --name vkg-test
multipass shell vkg-test
# Follow guide
```
