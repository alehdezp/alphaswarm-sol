## Step-by-Step Publishing

### Step 1: Clean Build

```bash
# Clean previous builds
rm -rf dist/ build/ *.egg-info

# Build package
uv build

# Verify build artifacts
ls dist/
# Should show:
# true_vkg-4.0.0.tar.gz
# true_vkg-4.0.0-py3-none-any.whl
```

### Step 2: Local Install Test

```bash
# Create fresh virtual environment
python -m venv /tmp/vkg-test
source /tmp/vkg-test/bin/activate

# Install from wheel
pip install dist/true_vkg-4.0.0-py3-none-any.whl

# Verify CLI works
vkg --version
# Expected: BSKG 4.0.0

# Verify import works
python -c "from true_vkg import KnowledgeGraph; print('OK')"

# Clean up
deactivate
rm -rf /tmp/vkg-test
```

### Step 3: Publish to Test PyPI

```bash
# Install twine if not available
pip install twine

# Upload to Test PyPI
twine upload --repository testpypi dist/*

# Credentials:
# Username: __token__
# Password: [Test PyPI API token from 16.0]
```

### Step 4: Test Install from Test PyPI

```bash
# Create fresh environment
python -m venv /tmp/vkg-test-pypi
source /tmp/vkg-test-pypi/bin/activate

# Install from Test PyPI
pip install --index-url https://test.pypi.org/simple/ \
            --extra-index-url https://pypi.org/simple/ \
            alphaswarm==4.0.0

# Verify
vkg --version
vkg --help

# Run a basic command
mkdir /tmp/vkg-demo && cd /tmp/vkg-demo
vkg init

# Clean up
deactivate
rm -rf /tmp/vkg-test-pypi /tmp/vkg-demo
```

### Step 5: Publish to Real PyPI

**Only after Test PyPI succeeds:**

```bash
# Upload to real PyPI
twine upload dist/*

# Credentials:
# Username: __token__
# Password: [Real PyPI API token from 16.0]
```

### Step 6: Verify Real PyPI Install

```bash
# Wait 2-5 minutes for PyPI to propagate

# Create fresh environment
python -m venv /tmp/vkg-final
source /tmp/vkg-final/bin/activate

# Install from PyPI
pip install alphaswarm==4.0.0

# Full verification
vkg --version
vkg --help

# Clean up
deactivate
rm -rf /tmp/vkg-final
```

## Troubleshooting

| Problem | Cause | Solution |
|---------|-------|----------|
| "Package not found" | Name conflict | Check package name on PyPI |
| "Invalid distribution" | Build issue | Clean and rebuild |
| "403 Forbidden" | Auth issue | Regenerate API token |
| "README rendering failed" | Markdown issue | Validate with `twine check dist/*` |
| Dependencies missing | Incomplete deps list | Add to pyproject.toml, rebuild |
