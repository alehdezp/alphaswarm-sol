## Test Protocol

### Ubuntu Tests (Docker - Fastest)

```bash
#!/bin/bash
# scripts/test_fresh_install_ubuntu.sh

set -e

echo "=== Testing BSKG Fresh Install on Ubuntu ==="

for PYTHON_VERSION in 3.11 3.10 3.9; do
    echo ""
    echo ">>> Testing Python $PYTHON_VERSION"
    echo ""

    docker run --rm -it python:${PYTHON_VERSION}-slim bash -c '
        set -e

        echo "Step 1: Install VKG"
        pip install alphaswarm

        echo "Step 2: Verify version"
        vkg --version

        echo "Step 3: Verify help"
        vkg --help

        echo "Step 4: Create test project"
        mkdir -p /tmp/test-project/contracts
        cat > /tmp/test-project/contracts/Test.sol << EOF
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract Test {
    mapping(address => uint256) public balances;

    function withdraw() public {
        uint256 amount = balances[msg.sender];
        (bool success, ) = msg.sender.call{value: amount}("");
        require(success);
        balances[msg.sender] = 0;
    }
}
EOF

        echo "Step 5: Initialize VKG"
        cd /tmp/test-project
        vkg init

        echo "Step 6: Build graph"
        vkg build contracts/

        echo "Step 7: Analyze"
        vkg analyze

        echo "Step 8: List findings"
        vkg findings list

        echo ""
        echo "=== SUCCESS: Python '"$PYTHON_VERSION"' passed ==="
    '
done

echo ""
echo "=== All Ubuntu tests passed ==="
```

### macOS Test

```bash
#!/bin/bash
# scripts/test_fresh_install_macos.sh

set -e

echo "=== Testing BSKG Fresh Install on macOS ==="

# Create fresh virtual environment
VENV_PATH="/tmp/vkg-macos-test"
rm -rf $VENV_PATH
python3 -m venv $VENV_PATH
source $VENV_PATH/bin/activate

# Install from PyPI
pip install alphaswarm

# Verify
vkg --version
vkg --help

# Test on sample project (if available)
if [ -d "./tests/projects/token-vault" ]; then
    cd ./tests/projects/token-vault
    rm -rf .vkg  # Clean any existing cache
    vkg init
    vkg build *.sol
    vkg analyze
    vkg findings list
fi

# Cleanup
deactivate
rm -rf $VENV_PATH

echo "=== macOS test passed ==="
```

### Windows Test (WSL or VM)

```powershell
# scripts/test_fresh_install_windows.ps1

Write-Host "=== Testing BSKG Fresh Install on Windows ==="

# Create fresh virtual environment
$VENV_PATH = "$env:TEMP\vkg-windows-test"
Remove-Item -Recurse -Force $VENV_PATH -ErrorAction SilentlyContinue
python -m venv $VENV_PATH
& "$VENV_PATH\Scripts\Activate.ps1"

# Install from PyPI
pip install alphaswarm

# Verify
vkg --version
vkg --help

# Cleanup
deactivate
Remove-Item -Recurse -Force $VENV_PATH

Write-Host "=== Windows test passed ==="
```

## Checklist Per Platform

### Must Work (All Platforms)

- [ ] `pip install alphaswarm` succeeds without errors
- [ ] `vkg --version` returns "VKG 4.0.0"
- [ ] `vkg --help` shows all commands
- [ ] `vkg init` creates .vrs/ directory
- [ ] `vkg build` succeeds on simple contract
- [ ] `vkg analyze` produces output
- [ ] `vkg findings list` works (even if empty)

### Should Work

- [ ] `vkg report --format sarif` produces valid SARIF
- [ ] Error messages are helpful (not stack traces)
- [ ] Exit codes are correct (0=success, non-zero=failure)

---

## Common Failure Points

| Issue | Detection | Fix |
|-------|-----------|-----|
| Slither not found | "command not found" | Verify Slither is in dependencies |
| solc not found | "solc: No such file" | Document solc-select or use foundry |
| PATH not set | "vkg: command not found" | Check pip install --user handling |
| Import error | Python exception | Missing dependency in pyproject.toml |
| Permission denied | OS error | Check file permissions, use temp dir |
| SSL errors | Certificate issues | Usually proxy/firewall, document |
