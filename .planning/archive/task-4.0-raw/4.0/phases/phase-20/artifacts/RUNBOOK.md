# Phase 20 Runbook

Log every run in this format.

## Run Entry

- run_id: RUN-001
- date: YYYY-MM-DD
- operator: <name>
- objective: <short>
- model: <model name>
- tools_available: [list]
- commands:
  - <command 1>
  - <command 2>
- inputs: <paths/urls>
- outputs: <paths>
- status: success|failure
- notes: <summary>
