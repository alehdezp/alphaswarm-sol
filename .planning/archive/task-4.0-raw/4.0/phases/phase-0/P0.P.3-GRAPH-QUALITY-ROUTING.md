## 1. Overview

This document defines **routing rules for graph quality issues** that escalate missing evidence, parse failures, and structural anomalies to a `graph-quality` convoy for systematic resolution.

**Key Distinction from P0.P.7:**
- **P0.P.7 (Tool Disagreement)**: Multiple tools produce conflicting *verdicts* about the same code
- **P0.P.3 (Graph Quality)**: The graph itself is incomplete, malformed, or missing expected data

**Philosophy Foundation (from `docs/PHILOSOPHY.md`):**
- "Every finding links to code locations and the IR/AST path that produced the signal"
- "Evidence from each tool is preserved and attributed"
- "If required fields are missing, set `request_more_context: true`"
- "Convoys group related beads for batch workflows and shared context"

**Key Constraint:** This specification is documentation only. No changes to `builder.py` are allowed. Graph quality detection and routing is a post-build operation applied during graph validation and finding assembly.

---

## 2. What Constitutes a Graph Quality Issue

### 2.1 Definition

A **graph quality issue** occurs when the knowledge graph cannot provide complete, accurate evidence for analysis. This is NOT about tool disagreement on findings; it's about the graph's ability to represent the code faithfully.

### 2.2 Graph Quality Issue Categories

| Category | ID | Description | Example |
|----------|-----|-------------|---------|
| **Parse Failure** | GQ-001 | Slither/parser could not parse source | Syntax errors, unsupported pragmas |
| **Incomplete Analysis** | GQ-002 | Partial analysis due to language features | Inline assembly, Yul blocks |
| **Proxy Resolution Failure** | GQ-003 | Could not resolve proxy implementation | Missing implementation address |
| **Missing Properties** | GQ-004 | Expected property not computed | Semantic op detection failed |
| **Graph Structure Anomaly** | GQ-005 | Orphan nodes, missing edges, cycles | Disconnected function nodes |
| **Build Failure** | GQ-006 | Compilation errors prevented analysis | Missing dependencies |
| **Evidence Gap** | GQ-007 | Property exists but lacks evidence | No source location for finding |
| **Signature Derivation Failure** | GQ-008 | Cannot compute behavioral signature | CFG unavailable or incomplete |

### 2.3 Graph Quality vs. Normal Incomplete Data

| Scenario | Classification | Routing |
|----------|---------------|---------|
| Function has no external calls (legitimately) | Normal | Standard processing |
| Function analysis skipped due to inline asm | GQ-002 | Graph-quality convoy |
| Property missing because function is simple | Normal | Standard processing |
| Property missing because parser failed | GQ-001 | Graph-quality convoy |
| No behavioral signature (simple function) | Normal | Standard processing |
| No behavioral signature (CFG parse failed) | GQ-008 | Graph-quality convoy |
| Tool ran but found nothing | Normal | Standard processing |
| Tool could not run due to build failure | GQ-006 | Graph-quality convoy |

### 2.4 Graph Quality Issue Severity Levels

| Severity | Impact | Threshold | Examples |
|----------|--------|-----------|----------|
| **CRITICAL** | Cannot analyze contract at all | Immediate escalation | GQ-001, GQ-006 (contract-level) |
| **HIGH** | Significant analysis gaps | >= 3 per contract | GQ-003, GQ-002 (multi-function) |
| **MEDIUM** | Partial analysis only | >= 5 per contract | GQ-004, GQ-005, GQ-008 |
| **LOW** | Minor evidence gaps | >= 10 per contract | GQ-007 |

#### 3.2.1 Parse Failure Detection (GQ-001)

```python
@dataclass
class ParseFailure:
    """Record of a parse failure."""

    file: str
    error_type: str       # syntax, pragma, import, encoding
    error_message: str
    line: int | None
    recoverable: bool     # True if partial parse succeeded

    def to_dict(self) -> dict:
        return {
            "issue_type": "GQ-001",
            "file": self.file,
            "error_type": self.error_type,
            "error_message": self.error_message,
            "line": self.line,
            "recoverable": self.recoverable,
            "severity": "CRITICAL" if not self.recoverable else "HIGH"
        }


def detect_parse_failures(
    slither_output: dict,
    compilation_output: dict
) -> list[ParseFailure]:
    """
    Detect parse failures from Slither and compilation output.

    Args:
        slither_output: Slither analysis output or error
        compilation_output: Solc compilation output

    Returns:
        List of ParseFailure records
    """
    failures = []

    # Check Slither errors
    if slither_output.get("error"):
        error = slither_output["error"]
        failures.append(ParseFailure(
            file=error.get("file", "unknown"),
            error_type="slither_parse",
            error_message=str(error.get("message", "Unknown error")),
            line=error.get("line"),
            recoverable=slither_output.get("partial_results") is not None
        ))

    # Check compilation errors
    for error in compilation_output.get("errors", []):
        if error.get("severity") == "error":
            failures.append(ParseFailure(
                file=error.get("sourceLocation", {}).get("file", "unknown"),
                error_type=error.get("type", "compilation"),
                error_message=error.get("message", "Unknown error"),
                line=error.get("sourceLocation", {}).get("start"),
                recoverable=False
            ))

    return failures
```

INCOMPLETE_ANALYSIS_FEATURES = {

    "yul_block": "Yul optimization blocks not analyzed",

    "external_library": "External library code not available",

def detect_incomplete_analysis(
    graph: "KnowledgeGraph",
    slither_analysis: dict
) -> list[IncompleteAnalysis]:
    """
    Detect functions with incomplete analysis.

    Args:
        graph: The built knowledge graph
        slither_analysis: Slither's analysis metadata

    Returns:
        List of IncompleteAnalysis records
    """
    incomplete = []

    for node in graph.nodes_by_type("Function"):
        props = node.properties

        # Check for inline assembly
        if props.get("has_inline_assembly"):
            incomplete.append(IncompleteAnalysis(
                file=props.get("file", "unknown"),
                contract=props.get("contract", "unknown"),
                function=node.label,
                feature="inline_assembly",
                lines=(props.get("line_start", 0), props.get("line_end", 0)),
                impact=INCOMPLETE_ANALYSIS_FEATURES["inline_assembly"]
            ))

        # Check for Yul blocks
        if props.get("has_yul_block"):
            incomplete.append(IncompleteAnalysis(
                file=props.get("file", "unknown"),
                contract=props.get("contract", "unknown"),
                function=node.label,
                feature="yul_block",
                lines=(props.get("line_start", 0), props.get("line_end", 0)),
                impact=INCOMPLETE_ANALYSIS_FEATURES["yul_block"]
            ))

        # Check for dynamic delegatecall
        if props.get("has_delegatecall") and not props.get("delegatecall_target_known"):
            incomplete.append(IncompleteAnalysis(
                file=props.get("file", "unknown"),
                contract=props.get("contract", "unknown"),
                function=node.label,
                feature="delegatecall_dynamic",
                lines=(props.get("line_start", 0), props.get("line_end", 0)),
                impact=INCOMPLETE_ANALYSIS_FEATURES["delegatecall_dynamic"]
            ))

    return incomplete

#### 3.2.3 Proxy Resolution Failure Detection (GQ-003)

@dataclass
class ProxyResolutionFailure:
    """Record of proxy resolution failure."""

    proxy_contract: str
    proxy_type: str           # eip1967, transparent, beacon, custom
    resolution_attempted: str  # What was tried
    failure_reason: str       # Why it failed
    fallback_used: str | None # What we fell back to

            "proxy_contract": self.proxy_contract,
            "proxy_type": self.proxy_type,
            "resolution_attempted": self.resolution_attempted,
            "failure_reason": self.failure_reason,
            "fallback_used": self.fallback_used,
            "severity": "HIGH"
        }


PROXY_RESOLUTION_METHODS = {
    "eip1967": "Read implementation slot 0x360894...",
    "transparent": "Check admin slot and implementation slot",
    "beacon": "Call beacon.implementation()",
    "uups": "Check ERC1822 implementation slot",
    "custom": "Pattern-based detection",
}


def detect_proxy_resolution_failures(
    graph: "KnowledgeGraph"
) -> list[ProxyResolutionFailure]:
    """
    Detect proxy contracts where implementation could not be resolved.

    Args:
        graph: The built knowledge graph

    Returns:
        List of ProxyResolutionFailure records
    """
    failures = []

    for node in graph.nodes_by_type("Contract"):
        props = node.properties

        if not props.get("is_proxy_like"):
            continue

        # Check if implementation was resolved
        if not props.get("implementation_resolved"):
            proxy_type = props.get("proxy_type", "unknown")
            failures.append(ProxyResolutionFailure(
                proxy_contract=node.label,
                proxy_type=proxy_type,
                resolution_attempted=PROXY_RESOLUTION_METHODS.get(proxy_type, "Unknown"),
                failure_reason=props.get("resolution_failure_reason", "Unknown"),
                fallback_used=props.get("resolution_fallback")
            ))

        # Check for partial resolution (implementation found but not analyzed)
        if (props.get("implementation_address") and
            not props.get("implementation_analyzed")):
            failures.append(ProxyResolutionFailure(
                proxy_contract=node.label,
                proxy_type=props.get("proxy_type", "unknown"),
                resolution_attempted="Implementation address found",
                failure_reason="Implementation contract code not available",
                fallback_used="Proxy-only analysis"
            ))

    return failures
```

            "source": "Call site analysis",
            "indicates": "External call tracking incomplete"
        },
        "behavioral_signature": {
            "condition": lambda p: len(p.get("semantic_operations", [])) > 0,
            "source": "Operation sequencing",

        },
        "has_access_gate": {

            "source": "Access control detection",
            "indicates": "Modifier analysis incomplete"
        },
    },
    "Contract": {
        "storage_layout": {
            "condition": lambda p: p.get("is_upgradeable"),
            "source": "Storage layout analysis",
            "indicates": "Upgrade safety analysis incomplete"
        },
    },
}


def detect_missing_properties(
    graph: "KnowledgeGraph"
) -> list[MissingProperty]:
    """
    Detect nodes missing expected properties.

    Args:
        graph: The built knowledge graph

    Returns:
        List of MissingProperty records
    """
    missing = []

    for node in graph.all_nodes():
        node_type = node.type
        props = node.properties

        expected = EXPECTED_PROPERTIES.get(node_type, {})

        for prop_name, spec in expected.items():
            # Check if property should exist based on condition
            should_exist = spec["condition"](props)

            if should_exist and prop_name not in props:
                missing.append(MissingProperty(
                    node_id=node.id,
                    node_type=node_type,
                    property_name=prop_name,
                    expected_source=spec["source"],
                    likely_cause=spec["indicates"]
                ))

    return missing
```

#### 3.2.5 Graph Structure Anomaly Detection (GQ-005)

```python
@dataclass
class StructureAnomaly:
    """Record of graph structure anomaly."""

    anomaly_type: str         # orphan_node, missing_edge, cycle, duplicate
    affected_nodes: list[str]
    description: str
    impact: str               # How this affects analysis

    def to_dict(self) -> dict:
        return {
            "issue_type": "GQ-005",
            "anomaly_type": self.anomaly_type,
            "affected_nodes": self.affected_nodes,
            "description": self.description,
            "impact": self.impact,
            "severity": "MEDIUM"
        }


def detect_structure_anomalies(
    graph: "KnowledgeGraph"
) -> list[StructureAnomaly]:
    """
    Detect structural anomalies in the graph.

    Args:
        graph: The built knowledge graph

    Returns:
        List of StructureAnomaly records
    """
    anomalies = []

    # Detect orphan Function nodes (no parent Contract)
    for node in graph.nodes_by_type("Function"):
        contract_edges = [
            e for e in graph.edges_from(node.id)
            if e.edge_type == "BELONGS_TO"
        ]
        if not contract_edges:
            anomalies.append(StructureAnomaly(
                anomaly_type="orphan_node",
                affected_nodes=[node.id],
                description=f"Function '{node.label}' has no parent contract",
                impact="Contract-level analysis may miss this function"
            ))

    # Detect missing CALLS edges
    for node in graph.nodes_by_type("Function"):
        props = node.properties
        if props.get("has_external_calls"):
            call_edges = [
                e for e in graph.edges_from(node.id)
                if e.edge_type == "CALLS"
            ]
            if not call_edges:
                anomalies.append(StructureAnomaly(
                    anomaly_type="missing_edge",
                    affected_nodes=[node.id],
                    description=f"Function '{node.label}' has external calls but no CALLS edges",
                    impact="Call graph incomplete; reentrancy analysis may miss paths"
                ))

    # Detect duplicate nodes (same contract.function)
    seen_labels = {}
    for node in graph.nodes_by_type("Function"):
        label_key = f"{node.properties.get('contract', '')}.{node.label}"
        if label_key in seen_labels:
            anomalies.append(StructureAnomaly(
                anomaly_type="duplicate",
                affected_nodes=[seen_labels[label_key], node.id],
                description=f"Duplicate function nodes for '{label_key}'",
                impact="Deduplication required; findings may be duplicated"
            ))
        else:
            seen_labels[label_key] = node.id

    # Detect cycles in inheritance (should be DAG)
    inheritance_cycles = detect_inheritance_cycles(graph)
    for cycle in inheritance_cycles:
        anomalies.append(StructureAnomaly(
            anomaly_type="cycle",
            affected_nodes=cycle,
            description=f"Inheritance cycle detected: {' -> '.join(cycle)}",
            impact="Inheritance analysis may be incorrect"
        ))

    return anomalies


def detect_inheritance_cycles(graph: "KnowledgeGraph") -> list[list[str]]:
    """Detect cycles in contract inheritance graph."""
    cycles = []
    visited = set()
    rec_stack = set()

    def dfs(node_id: str, path: list[str]) -> bool:
        visited.add(node_id)
        rec_stack.add(node_id)
        path.append(node_id)

        for edge in graph.edges_from(node_id):
            if edge.edge_type != "INHERITS":
                continue

            target = edge.target
            if target not in visited:
                if dfs(target, path):
                    return True
            elif target in rec_stack:
                # Found cycle
                cycle_start = path.index(target)
                cycles.append(path[cycle_start:] + [target])
                return True

        path.pop()
        rec_stack.remove(node_id)
        return False

    for node in graph.nodes_by_type("Contract"):
        if node.id not in visited:
            dfs(node.id, [])

    return cycles
```

#### 3.2.6 Build Failure Detection (GQ-006)

```python
@dataclass
class BuildFailure:
    """Record of build/compilation failure."""

    error_type: str           # import, version, dependency, syntax
    files_affected: list[str]
    error_message: str
    resolution_hint: str
    partial_build_possible: bool

    def to_dict(self) -> dict:
        return {
            "issue_type": "GQ-006",
            "error_type": self.error_type,
            "files_affected": self.files_affected,
            "error_message": self.error_message,
            "resolution_hint": self.resolution_hint,
            "partial_build_possible": self.partial_build_possible,
            "severity": "CRITICAL"
        }


BUILD_ERROR_HINTS = {
    "import": "Check import paths and ensure all dependencies are installed",
    "version": "Check pragma solidity version compatibility",
    "dependency": "Run 'forge install' or 'npm install' to install dependencies",
    "syntax": "Fix syntax errors before analysis",
    "encoding": "Check file encoding (UTF-8 expected)",
}


def detect_build_failures(
    compilation_result: dict
) -> list[BuildFailure]:
    """
    Detect build/compilation failures.

    Args:
        compilation_result: Result from solc compilation

    Returns:
        List of BuildFailure records
    """
    failures = []

    errors = compilation_result.get("errors", [])
    error_by_type = {}

    for error in errors:
        if error.get("severity") != "error":
            continue

        error_type = classify_build_error(error)
        if error_type not in error_by_type:
            error_by_type[error_type] = {
                "files": set(),
                "messages": []
            }

        file = error.get("sourceLocation", {}).get("file", "unknown")
        error_by_type[error_type]["files"].add(file)
        error_by_type[error_type]["messages"].append(error.get("message", ""))

    for error_type, data in error_by_type.items():
        failures.append(BuildFailure(
            error_type=error_type,
            files_affected=list(data["files"]),
            error_message="; ".join(data["messages"][:3]),  # First 3 messages
            resolution_hint=BUILD_ERROR_HINTS.get(error_type, "Check error message"),
            partial_build_possible=error_type in ("import", "dependency")
        ))

    return failures


def classify_build_error(error: dict) -> str:
    """Classify build error by type."""
    message = error.get("message", "").lower()
    error_type = error.get("type", "").lower()

    if "import" in message or "source not found" in message:
        return "import"
    if "pragma" in message or "version" in message:
        return "version"
    if "identifier not found" in message:
        return "dependency"
    if "parse" in error_type or "syntax" in message:
        return "syntax"

    return "unknown"
```

#### 3.2.7 Evidence Gap Detection (GQ-007)

```python
@dataclass
class EvidenceGap:
    """Record of property without evidence."""

    node_id: str
    property_name: str
    property_value: any
    expected_evidence: str    # What evidence should exist

    def to_dict(self) -> dict:
        return {
            "issue_type": "GQ-007",
            "node_id": self.node_id,
            "property_name": self.property_name,
            "property_value": str(self.property_value)[:50],
            "expected_evidence": self.expected_evidence,
            "severity": "LOW"
        }


# Properties that must have evidence
EVIDENCED_PROPERTIES = {
    "has_reentrancy_vulnerability": "Line numbers of external call and state write",
    "has_access_control_issue": "Line numbers of unprotected privileged write",
    "transfers_value": "Line numbers of value transfer",
    "reads_oracle": "Line numbers of oracle read",
    "has_unbounded_loop": "Line numbers of loop and array source",
}


def detect_evidence_gaps(
    graph: "KnowledgeGraph"
) -> list[EvidenceGap]:
    """
    Detect properties that exist without supporting evidence.

    Args:
        graph: The built knowledge graph

    Returns:
        List of EvidenceGap records
    """
    gaps = []

    for node in graph.all_nodes():
        props = node.properties
        evidence = node.evidence or []

        for prop_name, expected_evidence in EVIDENCED_PROPERTIES.items():
            if props.get(prop_name):
                # Property is set to true/non-null - check for evidence
                has_evidence = any(
                    e.detail and prop_name.lower() in e.detail.lower()
                    for e in evidence
                )

                if not has_evidence:
                    gaps.append(EvidenceGap(
                        node_id=node.id,
                        property_name=prop_name,
                        property_value=props[prop_name],
                        expected_evidence=expected_evidence
                    ))

    return gaps
```

#### 3.2.8 Signature Derivation Failure Detection (GQ-008)

```python
@dataclass
class SignatureDerivationFailure:
    """Record of behavioral signature derivation failure."""

    node_id: str
    function_name: str
    operations_detected: list[str]
    failure_reason: str
    cfg_available: bool

    def to_dict(self) -> dict:
        return {
            "issue_type": "GQ-008",
            "node_id": self.node_id,
            "function_name": self.function_name,
            "operations_detected": self.operations_detected,
            "failure_reason": self.failure_reason,
            "cfg_available": self.cfg_available,
            "severity": "MEDIUM"
        }


def detect_signature_failures(
    graph: "KnowledgeGraph"
) -> list[SignatureDerivationFailure]:
    """
    Detect functions where behavioral signature could not be derived.

    Args:
        graph: The built knowledge graph

    Returns:
        List of SignatureDerivationFailure records
    """
    failures = []

    for node in graph.nodes_by_type("Function"):
        props = node.properties

        # Skip simple functions (no operations expected)
        operations = props.get("semantic_operations", [])
        if not operations:
            continue

        # Check if signature was derived
        signature = props.get("behavioral_signature", "")

        if operations and not signature:
            # Has operations but no signature - derivation failed
            cfg_available = props.get("cfg_available", True)

            reason = "Unknown"
            if not cfg_available:
                reason = "CFG not available (inline assembly or complex control flow)"
            elif props.get("has_multiple_return_paths"):
                reason = "Multiple return paths; signature ambiguous"
            elif len(operations) > 10:
                reason = "Too many operations; signature too complex"

            failures.append(SignatureDerivationFailure(
                node_id=node.id,
                function_name=node.label,
                operations_detected=operations,
                failure_reason=reason,
                cfg_available=cfg_available
            ))

    return failures
```

## 4. The `graph-quality` Convoy Concept

### 4.1 What is a Graph-Quality Convoy?

A **graph-quality convoy** is a specialized batch of related graph issues that need systematic attention. Unlike finding convoys (which batch disputed findings), graph-quality convoys batch infrastructure-level problems that affect analysis quality.

**Purpose:**
1. **Batch Related Issues**: Group issues by root cause (e.g., all issues from one parse failure)
2. **Systematic Resolution**: Address root causes, not symptoms
3. **Prevent Cascade**: Stop bad graph data from producing false findings
4. **Track Technical Debt**: Monitor graph quality over time

### 4.2 Convoy Types

| Convoy Type | Trigger | Contents | Resolution Owner |
|-------------|---------|----------|------------------|
| `gq-parse` | Parse failures | GQ-001, GQ-006 | DevOps / Build system |
| `gq-analysis` | Analysis gaps | GQ-002, GQ-003 | BSKG Builder team |
| `gq-structure` | Structural issues | GQ-005, GQ-007 | Graph validation team |
| `gq-derivation` | Derivation failures | GQ-004, GQ-008 | Property detection team |
| `gq-mixed` | Multiple types | Mixed | Triage required |

### 4.3 Convoy Data Model

```python

class ConvoyStatus(Enum):

    DEFERRED = "deferred"         # Intentionally postponed

@dataclass

    id: str                               # convoy-gq-{timestamp}-{hash}
    convoy_type: str                      # gq-parse, gq-analysis, etc.
    issues: list[dict]                    # List of issue dicts
    affected_contracts: list[str]         # Contracts impacted
    affected_findings: list[str]          # Finding IDs impacted
    root_cause: str | None                # Identified root cause
    status: ConvoyStatus
    priority: str                         # CRITICAL, HIGH, MEDIUM, LOW
    created_at: str
    updated_at: str
    resolution: dict | None = None
    notes: list[str] = field(default_factory=list)

    @classmethod
    def create(
        cls,
        convoy_type: str,
        issues: list[dict],
        affected_contracts: list[str]
    ) -> "GraphQualityConvoy":
        """Create a new graph-quality convoy."""
        import hashlib

        timestamp = datetime.utcnow().strftime("%Y%m%dT%H%M%SZ")
        issue_ids = [i.get("issue_type", "") for i in issues]
        id_hash = hashlib.sha256(
            f"{convoy_type}:{timestamp}:{issue_ids}".encode()
        ).hexdigest()[:8]

        # Determine priority from highest severity issue
        severities = [i.get("severity", "LOW") for i in issues]
        priority = determine_convoy_priority(severities)

        return cls(
            id=f"convoy-gq-{id_hash}",
            convoy_type=convoy_type,
            issues=issues,
            affected_contracts=affected_contracts,
            affected_findings=[],  # Populated later
            root_cause=None,
            status=ConvoyStatus.PENDING,
            priority=priority,
            created_at=timestamp,
            updated_at=timestamp,
            resolution=None,
            notes=[]
        )

    def to_dict(self) -> dict:
        return {
            "id": self.id,
            "convoy_type": self.convoy_type,
            "issue_count": len(self.issues),
            "issues": self.issues,
            "affected_contracts": self.affected_contracts,
            "affected_findings": self.affected_findings,
            "root_cause": self.root_cause,
            "status": self.status.value,
            "priority": self.priority,
            "created_at": self.created_at,
            "updated_at": self.updated_at,
            "resolution": self.resolution,
            "notes": self.notes
        }

    def add_affected_finding(self, finding_id: str):
        """Track that a finding is affected by these graph quality issues."""
        if finding_id not in self.affected_findings:
            self.affected_findings.append(finding_id)
            self.updated_at = datetime.utcnow().isoformat()

    def set_root_cause(self, cause: str):
        """Set identified root cause."""
        self.root_cause = cause
        self.updated_at = datetime.utcnow().isoformat()

    def add_note(self, note: str):
        """Add investigation note."""
        timestamp = datetime.utcnow().strftime("%Y-%m-%d %H:%M")
        self.notes.append(f"[{timestamp}] {note}")
        self.updated_at = datetime.utcnow().isoformat()


def determine_convoy_priority(severities: list[str]) -> str:
    """Determine convoy priority from issue severities."""
    PRIORITY_ORDER = {"CRITICAL": 0, "HIGH": 1, "MEDIUM": 2, "LOW": 3}
    min_priority = min(PRIORITY_ORDER.get(s, 3) for s in severities)
    return {0: "CRITICAL", 1: "HIGH", 2: "MEDIUM", 3: "LOW"}[min_priority]
```

## 5. Convoy Creation Triggers

### 5.1 Trigger Thresholds

| Trigger ID | Condition | Convoy Type | Threshold |
|------------|-----------|-------------|-----------|
| CT-001 | Parse failure in any file | `gq-parse` | >= 1 |
| CT-002 | Build failure blocking analysis | `gq-parse` | >= 1 |
| CT-003 | Incomplete analysis (same feature) | `gq-analysis` | >= 3 functions |
| CT-004 | Proxy resolution failures | `gq-analysis` | >= 1 |
| CT-005 | Missing properties (same property) | `gq-derivation` | >= 5 nodes |
| CT-006 | Structure anomalies | `gq-structure` | >= 3 |
| CT-007 | Evidence gaps | `gq-structure` | >= 10 |
| CT-008 | Signature derivation failures | `gq-derivation` | >= 5 |
| CT-009 | Mixed issues in same contract | `gq-mixed` | >= 5 total |

### 5.2 Trigger Detection Function

```python
@dataclass
class ConvoyTrigger:
    """Record of a convoy creation trigger."""

    trigger_id: str
    condition_met: str
    issue_count: int
    convoy_type: str
    priority: str


def check_convoy_triggers(
    issues: dict[str, list[dict]],
    contract_name: str | None = None
) -> list[ConvoyTrigger]:
    """
    Check if any convoy triggers are met.

    Args:
        issues: Dict of issue_type -> list of issues
        contract_name: Optional contract filter

    Returns:
        List of ConvoyTrigger records for triggered thresholds
    """
    triggers = []

    # CT-001: Parse failure
    parse_failures = issues.get("GQ-001", [])
    if len(parse_failures) >= 1:
        triggers.append(ConvoyTrigger(
            trigger_id="CT-001",
            condition_met=f"Parse failures: {len(parse_failures)}",
            issue_count=len(parse_failures),
            convoy_type="gq-parse",
            priority="CRITICAL"
        ))

    # CT-002: Build failure
    build_failures = issues.get("GQ-006", [])
    if len(build_failures) >= 1:
        triggers.append(ConvoyTrigger(
            trigger_id="CT-002",
            condition_met=f"Build failures: {len(build_failures)}",
            issue_count=len(build_failures),
            convoy_type="gq-parse",
            priority="CRITICAL"
        ))

    # CT-003: Incomplete analysis (same feature)
    incomplete = issues.get("GQ-002", [])
    features = {}
    for i in incomplete:
        feat = i.get("feature", "unknown")
        features[feat] = features.get(feat, 0) + 1
    for feat, count in features.items():
        if count >= 3:
            triggers.append(ConvoyTrigger(
                trigger_id="CT-003",
                condition_met=f"Incomplete analysis ({feat}): {count} functions",
                issue_count=count,
                convoy_type="gq-analysis",
                priority="HIGH"
            ))

    # CT-004: Proxy resolution
    proxy_failures = issues.get("GQ-003", [])
    if len(proxy_failures) >= 1:
        triggers.append(ConvoyTrigger(
            trigger_id="CT-004",
            condition_met=f"Proxy resolution failures: {len(proxy_failures)}",
            issue_count=len(proxy_failures),
            convoy_type="gq-analysis",
            priority="HIGH"
        ))

    # CT-005: Missing properties (same property)
    missing_props = issues.get("GQ-004", [])
    props = {}
    for i in missing_props:
        prop = i.get("property_name", "unknown")
        props[prop] = props.get(prop, 0) + 1
    for prop, count in props.items():
        if count >= 5:
            triggers.append(ConvoyTrigger(
                trigger_id="CT-005",
                condition_met=f"Missing property '{prop}': {count} nodes",
                issue_count=count,
                convoy_type="gq-derivation",
                priority="MEDIUM"
            ))

    # CT-006: Structure anomalies
    anomalies = issues.get("GQ-005", [])
    if len(anomalies) >= 3:
        triggers.append(ConvoyTrigger(
            trigger_id="CT-006",
            condition_met=f"Structure anomalies: {len(anomalies)}",
            issue_count=len(anomalies),
            convoy_type="gq-structure",
            priority="MEDIUM"
        ))

    # CT-007: Evidence gaps
    evidence_gaps = issues.get("GQ-007", [])
    if len(evidence_gaps) >= 10:
        triggers.append(ConvoyTrigger(
            trigger_id="CT-007",
            condition_met=f"Evidence gaps: {len(evidence_gaps)}",
            issue_count=len(evidence_gaps),
            convoy_type="gq-structure",
            priority="LOW"
        ))

    # CT-008: Signature failures
    sig_failures = issues.get("GQ-008", [])
    if len(sig_failures) >= 5:
        triggers.append(ConvoyTrigger(
            trigger_id="CT-008",
            condition_met=f"Signature derivation failures: {len(sig_failures)}",
            issue_count=len(sig_failures),
            convoy_type="gq-derivation",
            priority="MEDIUM"
        ))

    # CT-009: Mixed issues in same contract
    if contract_name:
        total_issues = sum(len(v) for v in issues.values())
        if total_issues >= 5:
            issue_types = [k for k, v in issues.items() if len(v) > 0]
            if len(issue_types) >= 2:  # Multiple issue types
                triggers.append(ConvoyTrigger(
                    trigger_id="CT-009",
                    condition_met=f"Mixed issues in {contract_name}: {total_issues}",
                    issue_count=total_issues,
                    convoy_type="gq-mixed",
                    priority="HIGH"
                ))

    return triggers
```

### 6.1 Routing Decision Matrix

| Issue Type | Severity | Convoy Type | Routing Target | Priority |
|------------|----------|-------------|----------------|----------|
| GQ-001 | CRITICAL | `gq-parse` | Immediate convoy | CRITICAL |
| GQ-006 | CRITICAL | `gq-parse` | Immediate convoy | CRITICAL |
| GQ-003 | HIGH | `gq-analysis` | Convoy if >= 1 | HIGH |
| GQ-002 | HIGH/MEDIUM | `gq-analysis` | Convoy if >= 3 same feature | HIGH |
| GQ-005 | MEDIUM | `gq-structure` | Convoy if >= 3 | MEDIUM |
| GQ-008 | MEDIUM | `gq-derivation` | Convoy if >= 5 | MEDIUM |
| GQ-004 | MEDIUM | `gq-derivation` | Convoy if >= 5 same prop | MEDIUM |
| GQ-007 | LOW | `gq-structure` | Convoy if >= 10 | LOW |

### 6.2 Routing Function

```python
from enum import Enum


class GraphQualityRoute(Enum):
    CONVOY_IMMEDIATE = "convoy_immediate"   # Create convoy immediately
    CONVOY_BATCH = "convoy_batch"           # Add to batch, create convoy if threshold
    LOG_ONLY = "log_only"                   # Log issue, no convoy
    SUPPRESS = "suppress"                   # Known limitation, suppress


@dataclass
class RoutingDecision:
    """Routing decision for a graph quality issue."""

    route: GraphQualityRoute
    convoy_type: str | None
    priority: str
    reason: str
    convoy_id: str | None = None


def route_graph_quality_issue(
    issue: dict,
    existing_issues: dict[str, list[dict]],
    active_convoys: list[GraphQualityConvoy],
    config: dict
) -> RoutingDecision:
    """
    Determine routing for a graph quality issue.

    Args:
        issue: The issue dict
        existing_issues: Already detected issues by type
        active_convoys: Currently active convoys
        config: Configuration with suppression rules

    Returns:
        RoutingDecision
    """
    issue_type = issue.get("issue_type", "")
    severity = issue.get("severity", "MEDIUM")

    # Check for suppression (known limitations)
    suppressions = config.get("graph_quality_suppressions", [])
    for suppression in suppressions:
        if issue_matches_suppression(issue, suppression):
            return RoutingDecision(
                route=GraphQualityRoute.SUPPRESS,
                convoy_type=None,
                priority="NONE",
                reason=f"Suppressed by rule: {suppression.get('reason', 'Unknown')}"
            )

    # Critical issues -> immediate convoy
    if severity == "CRITICAL":
        convoy_type = "gq-parse"  # Critical issues are usually parse/build

        # Check for existing convoy to join
        for convoy in active_convoys:
            if convoy.convoy_type == convoy_type and convoy.status == ConvoyStatus.PENDING:
                return RoutingDecision(
                    route=GraphQualityRoute.CONVOY_IMMEDIATE,
                    convoy_type=convoy_type,
                    priority="CRITICAL",
                    reason="Joining existing critical convoy",
                    convoy_id=convoy.id
                )

        return RoutingDecision(
            route=GraphQualityRoute.CONVOY_IMMEDIATE,
            convoy_type=convoy_type,
            priority="CRITICAL",
            reason=f"Critical {issue_type} requires immediate convoy"
        )

    # Check convoy triggers
    all_issues = dict(existing_issues)
    if issue_type not in all_issues:
        all_issues[issue_type] = []
    all_issues[issue_type] = all_issues[issue_type] + [issue]

    triggers = check_convoy_triggers(all_issues)

    for trigger in triggers:
        if any(issue_type in trigger.condition_met for _ in [1]):
            # This issue contributes to a trigger
            return RoutingDecision(
                route=GraphQualityRoute.CONVOY_BATCH,
                convoy_type=trigger.convoy_type,
                priority=trigger.priority,
                reason=f"Threshold met: {trigger.condition_met}"
            )

    # Below threshold - log only
    return RoutingDecision(
        route=GraphQualityRoute.LOG_ONLY,
        convoy_type=None,
        priority=severity,
        reason=f"{issue_type} below convoy threshold"
    )


def issue_matches_suppression(issue: dict, suppression: dict) -> bool:
    """Check if issue matches a suppression rule."""
    # Match by issue type
    if suppression.get("issue_type") and suppression["issue_type"] != issue.get("issue_type"):
        return False

    # Match by file pattern
    if suppression.get("file_pattern"):
        import re
        file = issue.get("file", "")
        if not re.match(suppression["file_pattern"], file):
            return False

    # Match by feature
    if suppression.get("feature") and suppression["feature"] != issue.get("feature"):
        return False

    return True
```

## 7. Escalation Thresholds

### 7.1 Escalation Levels

| Level | Condition | Action | SLA |
|-------|-----------|--------|-----|
| **L0 - Log** | Below threshold | Log issue, continue | None |
| **L1 - Convoy** | Threshold met | Create convoy, batch process | 24 hours |
| **L2 - Priority Convoy** | HIGH severity | Priority convoy, notify team | 8 hours |
| **L3 - Immediate** | CRITICAL severity | Stop analysis, escalate | 2 hours |
| **L4 - Human** | Unresolved > SLA | Escalate to human review | Immediate |

### 7.2 Escalation Rules

```python
@dataclass
class EscalationDecision:
    """Escalation decision for convoy."""

    level: str                # L0, L1, L2, L3, L4
    action: str               # log, convoy, priority_convoy, immediate, human
    sla_hours: int | None
    notification_targets: list[str]
    blocking: bool            # Should analysis be blocked?


def determine_escalation(
    convoy: GraphQualityConvoy,
    current_time: datetime
) -> EscalationDecision:
    """
    Determine escalation level for convoy.

    Args:
        convoy: The convoy to evaluate
        current_time: Current timestamp

    Returns:
        EscalationDecision
    """
    # Check if SLA exceeded
    created = datetime.fromisoformat(convoy.created_at.replace("Z", "+00:00"))
    age_hours = (current_time - created).total_seconds() / 3600

    sla_by_priority = {
        "CRITICAL": 2,
        "HIGH": 8,
        "MEDIUM": 24,
        "LOW": 72,
    }

    sla = sla_by_priority.get(convoy.priority, 24)

    # L4: SLA exceeded
    if age_hours > sla and convoy.status not in (ConvoyStatus.RESOLVED, ConvoyStatus.WONT_FIX):
        return EscalationDecision(
            level="L4",
            action="human",
            sla_hours=None,
            notification_targets=["security-team", "dev-leads"],
            blocking=convoy.priority == "CRITICAL"
        )

    # L3: Critical priority
    if convoy.priority == "CRITICAL":
        return EscalationDecision(
            level="L3",
            action="immediate",
            sla_hours=2,
            notification_targets=["on-call", "security-team"],
            blocking=True
        )

    # L2: High priority
    if convoy.priority == "HIGH":
        return EscalationDecision(
            level="L2",
            action="priority_convoy",
            sla_hours=8,
            notification_targets=["security-team"],
            blocking=False
        )

    # L1: Standard convoy
    return EscalationDecision(
        level="L1",
        action="convoy",
        sla_hours=sla,
        notification_targets=[],
        blocking=False
    )
```

## 8. Integration with Finding Processing

### 8.1 Finding Impact Assessment

When a graph quality issue is detected, affected findings must be flagged:

```python
def assess_finding_impact(
    finding: dict,
    graph_issues: list[dict]
) -> dict:
    """
    Assess how graph quality issues impact a finding.

    Args:
        finding: The finding to assess
        graph_issues: List of detected graph quality issues

    Returns:
        Impact assessment dict
    """
    location = finding.get("location", {})
    file = location.get("file", "")
    contract = location.get("contract", "")
    function = location.get("function", "")

    impacts = []

    for issue in graph_issues:
        issue_type = issue.get("issue_type", "")

        # Check if issue affects this finding's location
        if issue_type in ("GQ-001", "GQ-006"):
            # Parse/build failures affect entire file
            if issue.get("file") == file or file in issue.get("files_affected", []):
                impacts.append({
                    "issue_type": issue_type,
                    "impact": "Finding may be based on incomplete parse",
                    "severity": "HIGH",
                    "recommendation": "Re-analyze after fixing parse/build errors"
                })

        elif issue_type == "GQ-002":
            # Incomplete analysis affects specific function
            if (issue.get("contract") == contract and
                issue.get("function") == function):
                impacts.append({
                    "issue_type": issue_type,
                    "impact": f"Function analysis incomplete due to {issue.get('feature')}",
                    "severity": "MEDIUM",
                    "recommendation": "Manual review of inline assembly/Yul"
                })

        elif issue_type == "GQ-003":
            # Proxy resolution affects entire proxy contract
            if issue.get("proxy_contract") == contract:
                impacts.append({
                    "issue_type": issue_type,
                    "impact": "Proxy implementation not resolved",
                    "severity": "HIGH",
                    "recommendation": "Provide implementation address or analyze implementation separately"
                })

        elif issue_type == "GQ-008":
            # Signature failure affects specific function
            if issue.get("function_name") == function:
                impacts.append({
                    "issue_type": issue_type,
                    "impact": "Behavioral signature not available",
                    "severity": "MEDIUM",
                    "recommendation": "Finding relies on property-based detection only"
                })

    # Determine overall impact
    if not impacts:
        return {
            "affected": False,
            "impacts": [],
            "overall_severity": None,
            "confidence_penalty": 0.0
        }

    severities = [i["severity"] for i in impacts]
    overall = "HIGH" if "HIGH" in severities else "MEDIUM" if "MEDIUM" in severities else "LOW"

    # Compute confidence penalty
    penalty = 0.0
    for impact in impacts:
        if impact["severity"] == "HIGH":
            penalty += -0.15
        elif impact["severity"] == "MEDIUM":
            penalty += -0.10
        else:
            penalty += -0.05

    penalty = max(penalty, -0.30)  # Cap at -0.30

    return {
        "affected": True,
        "impacts": impacts,
        "overall_severity": overall,
        "confidence_penalty": penalty,
        "recommendation": impacts[0]["recommendation"] if impacts else None
    }
```

### 8.2 Evidence Packet Integration

When graph issues affect a finding, the evidence packet is updated:

```python
def update_packet_for_graph_issues(
    packet: dict,
    impact_assessment: dict,
    convoy_id: str | None
) -> dict:
    """
    Update evidence packet with graph quality impact information.

    Args:
        packet: The evidence packet
        impact_assessment: Result from assess_finding_impact()
        convoy_id: Associated graph-quality convoy ID (if any)

    Returns:
        Updated packet
    """
    if not impact_assessment.get("affected"):
        return packet

    # Add graph quality metadata
    packet["graph_quality_issues"] = {
        "affected": True,
        "impacts": impact_assessment["impacts"],
        "overall_severity": impact_assessment["overall_severity"],
        "convoy_id": convoy_id,
    }

    # Apply confidence penalty
    current_score = packet.get("confidence_score", 0.5)
    new_score = max(0.0, current_score + impact_assessment["confidence_penalty"])
    packet["confidence_score"] = new_score

    # Update bucket if needed
    if new_score < 0.5 and packet.get("confidence_bucket") == "likely":
        packet["confidence_bucket"] = "uncertain"
        packet["bucket_rationale"] = (
            f"{packet.get('bucket_rationale', '')}; "
            f"Downgraded due to graph quality issues ({impact_assessment['overall_severity']})"
        )

    # Set request_more_context for HIGH impact
    if impact_assessment["overall_severity"] == "HIGH":
        packet["request_more_context"] = True
        packet["context_hint"] = (
            f"Graph quality issue: {impact_assessment['impacts'][0]['impact']}"
        )

    return packet
```
