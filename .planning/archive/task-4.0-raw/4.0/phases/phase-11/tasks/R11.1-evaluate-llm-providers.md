## Evaluation Criteria

### 1. Cost Analysis

Create spreadsheet comparing:
- Cost per 1M input tokens
- Cost per 1M output tokens
- Typical cost per finding verification (estimate 2K input, 500 output)
- Monthly budget at 1000 findings/day

### 2. Quality Benchmarks

For each provider, test on 20 sample findings:
- 10 TRUE POSITIVES (real vulnerabilities)
- 10 FALSE POSITIVES (safe code flagged by patterns)

Measure:
- **Accuracy:** Correctly identified TP as vulnerable, FP as safe
- **Reasoning quality:** Evidence cited, logical explanation
- **Hallucination rate:** Made up facts or code references
- **Structured output compliance:** Follows JSON schema

### 3. Latency Benchmarks

Measure for typical 2K token input:
- P50 latency
- P95 latency
- Timeout rate (> 30s)

### 4. Reliability

Track over 24h test period:
- Uptime percentage
- Rate limit hits
- Error rates
- Recovery behavior

### Test Script

Create `scripts/benchmark_llm_providers.py`:

```python
"""Benchmark LLM providers for Tier B analysis."""

import asyncio
import json
import time
from dataclasses import dataclass, asdict
from pathlib import Path
from typing import List, Dict

@dataclass
class BenchmarkResult:
    provider: str
    model: str
    finding_id: str
    correct: bool
    latency_ms: int
    tokens_in: int
    tokens_out: int
    cost_usd: float
    verdict: str
    expected_verdict: str
    hallucination: bool
    schema_valid: bool

async def benchmark_provider(provider, model, findings):
    results = []
    for finding in findings:
        start = time.monotonic()
        response = await provider.analyze(finding)
        latency = int((time.monotonic() - start) * 1000)

        result = BenchmarkResult(
            provider=provider.name,
            model=model,
            finding_id=finding["id"],
            correct=response.verdict == finding["expected_verdict"],
            latency_ms=latency,
            tokens_in=response.input_tokens,
            tokens_out=response.output_tokens,
            cost_usd=response.cost_usd,
            verdict=response.verdict,
            expected_verdict=finding["expected_verdict"],
            hallucination=detect_hallucination(response, finding),
            schema_valid=validate_schema(response),
        )
        results.append(result)
    return results

def main():
    # Load findings
    findings = json.loads(Path("benchmark_findings.json").read_text())

    # Benchmark each provider
    all_results = []
    for provider_config in PROVIDERS:
        results = asyncio.run(benchmark_provider(...))
        all_results.extend(results)

    # Generate report
    generate_report(all_results)
```
