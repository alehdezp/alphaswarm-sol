## Objective

---

## What This Solution Does

**Location:** `src/true_vkg/adversarial/`
**Tests:** 37 tests in `tests/test_adversarial.py`

The Adversarial Test Generation solution uses LLMs to generate test cases:
1. Analyzes contract coverage gaps
2. Generates Solidity test cases targeting uncovered paths
3. Aims for 85%+ coverage on difficult contracts
4. Produces compilable, runnable tests

**Claimed Benefits:**
- Automated test generation
- Finds edge cases humans miss
- Targets coverage gaps intelligently

**Cost Concern:** Uses LLM API calls = ongoing costs
