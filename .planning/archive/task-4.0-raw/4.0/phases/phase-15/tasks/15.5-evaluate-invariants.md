## What This Solution Does

**Location:** `src/true_vkg/invariants/`
**Tests:** 47 tests in `tests/test_invariants.py`

The Formal Invariant Synthesis solution:
1. Analyzes contract code and NatSpec comments
2. Synthesizes formal invariants (properties that should always hold)
3. Uses Z3/SMT to verify invariants
4. Generates invariant violations as findings

**Claimed Benefits:**
- Automatic formal property extraction
- Mathematical verification via Z3
- Catches subtle bugs that testing misses

**Dependency Concern:** Requires z3-solver (heavy dependency)
