## Red Flags to Watch For

1. **Overfitting**: High accuracy on training data, poor on new contracts
2. **Data Leakage**: Using future information to predict past events
3. **Unvalidated Claims**: 75% accuracy with no reproducible benchmark
4. **Heavy Dependencies**: sklearn/scipy add significant install size
5. **Stale Model**: Predictions based on outdated patterns
