## What This Solution Does

**Location:** `src/true_vkg/evolution/`
**Tests:** 27 tests in `tests/test_evolution.py`

The Self-Evolving Patterns solution uses genetic algorithms to optimize pattern F1 scores:
1. Patterns are encoded as chromosomes
2. Genetic operators (mutation, crossover) create variations
3. Fitness function measures F1 score on test cases
4. Best patterns survive and reproduce
5. Over generations, patterns improve

**Claimed Benefits:**
- Automatic pattern optimization
- Discovers edge cases humans miss
- Continuous improvement over time
