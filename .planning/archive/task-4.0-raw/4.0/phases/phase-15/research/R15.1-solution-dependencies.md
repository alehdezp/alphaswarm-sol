## Research Protocol

For each solution, determine:

### 1. Python Dependencies

```bash
# For each solution, check imports
grep -r "^import\|^from" src/true_vkg/{solution}/ | grep -v "true_vkg" | sort | uniq
```

**Record:**
- External packages required
- Version constraints (if any)
- Whether in current pyproject.toml

### 2. External API Dependencies

Check for:
- LLM API calls (Anthropic, OpenAI, etc.)
- Network calls (web APIs, databases)
- External services

**Questions:**
- Does it work offline?
- What API keys are needed?
- What's the cost per operation?

### 3. Internal BSKG Dependencies

```bash
# Check which BSKG modules are imported
grep -r "from true_vkg" src/true_vkg/{solution}/ | sort | uniq
```

**Record:**
- Which core modules are required
- Coupling level (tight vs loose)
- Could break if core changes

### 4. Optional vs Required

For each dependency, classify:
- **REQUIRED**: Won't work without it
- **OPTIONAL**: Works with reduced functionality
- **DEV_ONLY**: Only for testing
