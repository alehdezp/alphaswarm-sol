## Implementation

Create `src/true_vkg/learning/label_validator.py`:

Validation checks:
- Evidence line numbers exist in `CodeSnippet`
- Label in allowlist
- Confidence >= threshold
- Category relevance (label must match candidate category)
- Guard labels must match detected guard patterns

Example:
```python
if label == "IS_REENTRANCY_GUARD":
    assert "REENTRANCY_GUARD" in extract_guards(code)
```
