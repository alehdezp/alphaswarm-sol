## Objective

Implement a microagent that assigns semantic labels from a compact
context slice using a strict output schema and token budget.

Relevant modules:

- `src/true_vkg/learning/overlay.py`

---

## Implementation

Create `src/true_vkg/agents/labeler_agent.py`:

- `LabelerMicroAgent` extends `MicroAgent`
- Input: `LabelCandidate` + minimal code/context
- Output schema (JSON):
  - `labels`: list of `{label, node_id, confidence, evidence_lines}`
  - `edges`: list of `{relation, source_id, target_id, confidence, evidence_lines}`
  - `notes`: optional strings (ignored by parser)

Prompt constraints:
- Max 6k tokens total input
- No prose in output
- Every label must reference evidence lines

Add parser in `src/true_vkg/learning/labeler.py`:
- Validate JSON
- Enforce label allowlist
- Convert to `LearningAssertion` candidates
