## Objective

Enable pattern matching against label overlays without modifying
`builder.py` or `executor.py`.

---

## Context

Pattern engine:
- `src/true_vkg/queries/patterns.py`

Overlay:
- `src/true_vkg/learning/overlay.py`

---

## Implementation

Create `src/true_vkg/queries/label_patterns.py`:

- Build a **LabelAwareGraphView** that wraps `KnowledgeGraph`
  and injects a `labels` property per node:
  - `node.properties["labels"] = ["IS_ACCESS_GATE", ...]`

- Use existing operators:
  - `property: labels`, `op: contains_any`, `value: ["IS_ACCESS_GATE"]`

- Provide a new entrypoint:
  - `run_label_aware_patterns(graph, patterns, overlay_store)`

Avoid changes to `executor.py` by adding a separate optional
call in CLI or analysis flow.
