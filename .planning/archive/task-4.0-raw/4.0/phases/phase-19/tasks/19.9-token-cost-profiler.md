**Status:** TODO
**Priority:** SHOULD
**Estimated Hours:** 6h
**Depends On:** 19.2

---

## Objective

Track token usage and cost for the labeler pipeline, enforce
per-call budgets, and provide an audit trail for optimization.

---

## Context

Relevant modules:
- `src/true_vkg/agents/cost.py` (CostTracker + pricing)
- `src/true_vkg/agents/microagent.py` (token tracking)
- `src/true_vkg/learning/labeler.py` (labeler entrypoint)

---

## Implementation

1. Add a labeler-specific cost profiler:
   - `src/true_vkg/learning/cost_profiler.py`
   - Wraps `CostTracker` with labeler-specific tags
   - Records token counts, duration, and bead/task context

2. Wire the profiler into labeler execution:
   - Record input/output token counts per call
   - Enforce soft budget (warn) at 6k tokens
   - Enforce hard budget (reject) at 8k tokens

3. Persist reports per run:
   - `.vrs/learning/costs.jsonl` (one record per call)
   - Aggregated report: `.vrs/learning/cost-report.json`

4. Add a CLI helper:
   - `alphaswarm learn cost report`
   - Prints summary from `CostTracker.get_report()`
