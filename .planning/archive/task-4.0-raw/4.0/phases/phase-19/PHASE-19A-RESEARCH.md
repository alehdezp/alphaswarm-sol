# Phase 19A: Research Foundations

**Purpose:** Gather the knowledge needed to design a label taxonomy and
labeler pipeline that can detect complex, business-logic vulnerabilities.

---

## Scope

- Define what label classes are needed (intent, capability, authority, state).
- Define how to mine specs and business rules from docs and code.
- Set LLM labeler budget constraints and prompt schema.
- Select real-world evaluation corpus and ground truth plan.
