## 1. OBJECTIVE

Create a task router that selects the appropriate LLM provider and model tier based on task type. This is the core routing logic WITHOUT batch dispatch or TOON encoding.

**Scope:** Only the routing/selection logic, not execution.

## 2. CONTEXT

### 2.1 Why This Matters

Different LLM tasks have different requirements:

| Task Type | Complexity | Best Tier | Example Model |
|-----------|------------|-----------|---------------|
| Evidence extraction | Low | Cheap | claude-3-haiku |
| Test generation | Medium | Medium | claude-sonnet |
| Exploit synthesis | High | Expensive | claude-opus |

Routing ensures cost-efficiency without sacrificing quality.

```python
from dataclasses import dataclass
from enum import Enum
from typing import Optional, Dict, Any

class TaskType(Enum):
    """Types of LLM tasks for routing."""
    EVIDENCE_EXTRACTION = "evidence_extraction"
    CODE_SUMMARIZATION = "code_summarization"
    FINDING_DEDUP = "finding_dedup"
    TIER_B_VERIFICATION = "tier_b_verification"
    TEST_SCAFFOLD_GENERATION = "test_scaffold_generation"
    FALSE_POSITIVE_ANALYSIS = "false_positive_analysis"
    EXPLOIT_SYNTHESIS = "exploit_synthesis"
    MULTI_CONTRACT_ANALYSIS = "multi_contract_analysis"

class ModelTier(Enum):
    """Model tier for cost/quality tradeoff."""
    CHEAP = "cheap"      # Fast, low-cost: haiku, gpt-4o-mini
    MEDIUM = "medium"    # Balanced: sonnet, gpt-4o
    EXPENSIVE = "expensive"  # Best: opus, o1

@dataclass
class SubagentTask:
    """Task to be executed by an LLM subagent."""
    task_type: TaskType
    prompt: str
    context: Dict[str, Any]  # Context data (encoding done later)
    output_schema: Optional[Dict] = None  # Expected JSON schema
    preferred_provider: Optional[str] = None  # Override default
    preferred_tier: Optional[ModelTier] = None  # Override default
    max_cost_usd: float = 0.50
```

### 3.3 Router Interface

```python
@dataclass
class RoutingDecision:
    """Result of routing a task to a provider/model."""
    provider: str  # e.g., "anthropic", "openai"
    model: str     # e.g., "claude-3-haiku-20240307"
    tier: ModelTier
    estimated_cost_per_1k_tokens: float
    reason: str    # Why this provider/model was chosen

class TaskRouter:
    """Routes LLM tasks to appropriate provider and model."""

    def __init__(self, config: LLMConfig):
        """
        Initialize router with LLM configuration.

        Args:
            config: LLM configuration with available providers
        """
        self.config = config
        self.task_tiers = TASK_TIER_DEFAULTS.copy()

    def route(self, task: SubagentTask) -> RoutingDecision:
        """
        Route a task to the best provider/model.

        Priority:
        1. Task's preferred_provider and preferred_tier (if specified)
        2. Task type default tier from TASK_TIER_DEFAULTS
        3. Config default provider and tier

        Args:
            task: The task to route

        Returns:
            RoutingDecision with selected provider/model
        """
        pass  # Implementation below

    def override_task_tier(self, task_type: TaskType, tier: ModelTier):
        """Override the default tier for a task type."""
        self.task_tiers[task_type] = tier
```

```python
# Key implementation detail
def route(self, task: SubagentTask) -> RoutingDecision:
    # Step 1: Determine tier
    if task.preferred_tier:
        tier = task.preferred_tier
    else:
        tier = self.task_tiers.get(task.task_type, self.config.default_tier)

    # Step 2: Determine provider
    if task.preferred_provider and task.preferred_provider in self._available_providers:
        provider_name = task.preferred_provider
    else:
        provider_name = self.config.default_provider

    # Step 3: Get model for tier
    provider_config = self.config.providers[provider_name]
    model = provider_config.models[tier]

    # Step 4: Get cost estimate
    cost = self._get_cost_per_1k(provider_name, model)

    return RoutingDecision(
        provider=provider_name,
        model=model,
        tier=tier,
        estimated_cost_per_1k_tokens=cost,
        reason=f"Task type {task.task_type.value} -> tier {tier.value}"
    )
```
