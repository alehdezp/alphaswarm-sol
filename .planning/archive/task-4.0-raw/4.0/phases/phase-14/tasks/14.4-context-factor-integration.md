## Objective

Adjust calibrated confidence based on contextual factors extracted from the knowledge graph, such as presence of guards, access control, or trusted external calls.

---

## Why Context Factors?

Calibration alone uses pattern-level statistics. But individual findings have context:

- A reentrancy finding with `has_reentrancy_guard=true` is less likely to be exploitable
- An access control finding where the function already has `onlyOwner` modifier is likely FP
- A finding on a function calling a trusted contract (e.g., Uniswap) may be less risky

Context factors adjust the calibrated confidence based on this function-specific information.

## Context Factors to Implement

### Factor 1: Reentrancy Guard

**Source:** `has_reentrancy_guard` property on function node
**Effect:** Significantly reduces reentrancy vulnerability confidence
**Multiplier:** 0.3 (reduces to 30% of calibrated value)

**Logic:**
```python
if finding.pattern_id.startswith("vm-") and context.get("has_reentrancy_guard"):
    adjusted *= 0.3
```

### Factor 2: Access Control Present

**Source:** `has_access_gate` property
**Effect:** Reduces access control vulnerability confidence
**Multiplier:** 0.7

**Logic:**
```python
if finding.pattern_id.startswith("auth-") and context.get("has_access_gate"):
    adjusted *= 0.7
```

### Factor 3: Trusted External Call

**Source:** External call targets known safe contracts
**Effect:** Reduces exploitation confidence
**Multiplier:** 0.5

**Logic:**
```python
TRUSTED_CONTRACTS = ["uniswap", "aave", "compound", "chainlink"]
if any(trusted in context.get("external_call_targets", []) for trusted in TRUSTED_CONTRACTS):
    adjusted *= 0.5
```

### Factor 4: SafeERC20 Usage

**Source:** `uses_safe_erc20` property
**Effect:** Reduces token handling vulnerability confidence
**Multiplier:** 0.4

### Factor 5: CEI Pattern Detected

**Source:** `follows_cei_pattern` property
**Effect:** Reduces reentrancy confidence further
**Multiplier:** 0.2

### Step 1: Create Context Adjuster (2.5h)

Create: `src/true_vkg/calibration/context.py`

```python
from dataclasses import dataclass, field
from typing import Dict, List, Optional

@dataclass
class ContextFactors:
    """Contextual factors extracted from knowledge graph."""
    has_reentrancy_guard: bool = False
    has_access_gate: bool = False
    uses_safe_erc20: bool = False
    follows_cei_pattern: bool = False
    external_call_targets: List[str] = field(default_factory=list)
    has_onchain_validation: bool = False
    uses_openzeppelin: bool = False

    @classmethod
    def from_function_node(cls, function_node: dict) -> "ContextFactors":
        """Extract context factors from a function node."""
        return cls(
            has_reentrancy_guard=function_node.get("has_reentrancy_guard", False),
            has_access_gate=function_node.get("has_access_gate", False),
            uses_safe_erc20=function_node.get("uses_safe_erc20", False),
            follows_cei_pattern=function_node.get("follows_cei_pattern", False),
            external_call_targets=function_node.get("external_call_targets", []),
            has_onchain_validation=function_node.get("has_onchain_validation", False),
            uses_openzeppelin=function_node.get("uses_openzeppelin", False),
        )

@dataclass
class ContextAdjustmentConfig:
    """Configuration for context-based adjustments."""
    reentrancy_guard_multiplier: float = 0.3
    access_gate_multiplier: float = 0.7
    safe_erc20_multiplier: float = 0.4
    cei_pattern_multiplier: float = 0.2
    trusted_call_multiplier: float = 0.5
    openzeppelin_multiplier: float = 0.8

    # Caps to prevent extreme adjustments
    min_confidence: float = 0.05  # Never go below 5%
    max_confidence: float = 0.98  # Never go above 98%

    # Trusted contract patterns
    trusted_patterns: List[str] = field(default_factory=lambda: [
        "uniswap", "sushiswap", "aave", "compound",
        "chainlink", "openzeppelin", "maker"
    ])

class ContextAdjuster:
    """Adjusts calibrated confidence based on contextual factors."""

    def __init__(self, config: Optional[ContextAdjustmentConfig] = None):
        self.config = config or ContextAdjustmentConfig()

    def adjust(
        self,
        calibrated_confidence: float,
        pattern_id: str,
        context: ContextFactors
    ) -> tuple[float, List[str]]:
        """
        Adjust calibrated confidence based on context.

        Args:
            calibrated_confidence: Score from calibration model
            pattern_id: The pattern that triggered the finding
            context: Contextual factors from KG

        Returns:
            Tuple of (adjusted_confidence, list of adjustments applied)
        """
        adjusted = calibrated_confidence
        adjustments_applied = []

        # Factor 1: Reentrancy guard for reentrancy patterns
        if self._is_reentrancy_pattern(pattern_id) and context.has_reentrancy_guard:
            adjusted *= self.config.reentrancy_guard_multiplier
            adjustments_applied.append(
                f"reentrancy_guard: *{self.config.reentrancy_guard_multiplier}"
            )

        # Factor 2: CEI pattern for reentrancy patterns
        if self._is_reentrancy_pattern(pattern_id) and context.follows_cei_pattern:
            adjusted *= self.config.cei_pattern_multiplier
            adjustments_applied.append(
                f"cei_pattern: *{self.config.cei_pattern_multiplier}"
            )

        # Factor 3: Access gate for access control patterns
        if self._is_access_control_pattern(pattern_id) and context.has_access_gate:
            adjusted *= self.config.access_gate_multiplier
            adjustments_applied.append(
                f"access_gate: *{self.config.access_gate_multiplier}"
            )

        # Factor 4: SafeERC20 for token patterns
        if self._is_token_pattern(pattern_id) and context.uses_safe_erc20:
            adjusted *= self.config.safe_erc20_multiplier
            adjustments_applied.append(
                f"safe_erc20: *{self.config.safe_erc20_multiplier}"
            )

        # Factor 5: Trusted external calls
        if self._has_trusted_call(context.external_call_targets):
            adjusted *= self.config.trusted_call_multiplier
            adjustments_applied.append(
                f"trusted_call: *{self.config.trusted_call_multiplier}"
            )

        # Factor 6: OpenZeppelin usage
        if context.uses_openzeppelin:
            adjusted *= self.config.openzeppelin_multiplier
            adjustments_applied.append(
                f"openzeppelin: *{self.config.openzeppelin_multiplier}"
            )

        # Apply caps
        adjusted = max(self.config.min_confidence, min(self.config.max_confidence, adjusted))

        return adjusted, adjustments_applied

    def _is_reentrancy_pattern(self, pattern_id: str) -> bool:
        return pattern_id.startswith("vm-") or "reentrancy" in pattern_id.lower()

    def _is_access_control_pattern(self, pattern_id: str) -> bool:
        return pattern_id.startswith("auth-") or "access" in pattern_id.lower()

    def _is_token_pattern(self, pattern_id: str) -> bool:
        return pattern_id.startswith("token-") or "erc20" in pattern_id.lower()

    def _has_trusted_call(self, targets: List[str]) -> bool:
        for target in targets:
            target_lower = target.lower()
            if any(trusted in target_lower for trusted in self.config.trusted_patterns):
                return True
        return False
```

### Step 2: Integrate with Calibrator Pipeline (1.5h)

Create: `src/true_vkg/calibration/pipeline.py`

```python
"""Complete calibration pipeline combining all components."""

from dataclasses import dataclass
from typing import Optional, List
from pathlib import Path

from .calibrator import ConfidenceCalibrator
from .pattern_calibrator import PatternCalibrator
from .context import ContextAdjuster, ContextFactors, ContextAdjustmentConfig

@dataclass
class CalibrationResult:
    """Result of confidence calibration."""
    raw_confidence: float
    calibrated_confidence: float
    context_adjusted_confidence: float
    final_confidence: float  # Same as context_adjusted for now
    calibrator_used: str  # "pattern:vm-001" or "global"
    context_adjustments: List[str]

class CalibrationPipeline:
    """
    Complete pipeline for confidence calibration.

    1. Raw confidence from pattern match
    2. Calibrated via pattern or global calibrator
    3. Adjusted based on context factors
    """

    def __init__(
        self,
        calibrator: Optional[PatternCalibrator] = None,
        context_adjuster: Optional[ContextAdjuster] = None
    ):
        self.calibrator = calibrator
        self.context_adjuster = context_adjuster or ContextAdjuster()

    @classmethod
    def load(cls, calibrator_path: Path) -> "CalibrationPipeline":
        """Load pipeline with calibrator from disk."""
        calibrator = PatternCalibrator.load(calibrator_path)
        return cls(calibrator=calibrator)

    def calibrate(
        self,
        raw_confidence: float,
        pattern_id: str,
        context: Optional[ContextFactors] = None
    ) -> CalibrationResult:
        """
        Run full calibration pipeline.

        Args:
            raw_confidence: Raw confidence from pattern match
            pattern_id: Pattern ID
            context: Optional context factors from KG

        Returns:
            CalibrationResult with all stages
        """
        # Stage 1: Pattern/Global calibration
        if self.calibrator is not None:
            calibrated, calibrator_used = self.calibrator.calibrate(pattern_id, raw_confidence)
        else:
            calibrated = raw_confidence
            calibrator_used = "none"

        # Stage 2: Context adjustment
        if context is not None:
            context_adjusted, adjustments = self.context_adjuster.adjust(
                calibrated, pattern_id, context
            )
        else:
            context_adjusted = calibrated
            adjustments = []

        return CalibrationResult(
            raw_confidence=raw_confidence,
            calibrated_confidence=calibrated,
            context_adjusted_confidence=context_adjusted,
            final_confidence=context_adjusted,
            calibrator_used=f"pattern:{calibrator_used}" if calibrator_used != "global" else "global",
            context_adjustments=adjustments
        )
```

from true_vkg.calibration.context import (
    ContextFactors, ContextAdjuster, ContextAdjustmentConfig
)
from true_vkg.calibration.pipeline import CalibrationPipeline, CalibrationResult

class TestContextAdjuster:
    """Tests for ContextAdjuster."""

    def test_reentrancy_guard_reduces_confidence(self):
        """Reentrancy guard should reduce reentrancy finding confidence."""
        adjuster = ContextAdjuster()
        context = ContextFactors(has_reentrancy_guard=True)

        adjusted, _ = adjuster.adjust(0.8, "vm-001", context)

        assert adjusted < 0.8 * 0.5  # Significant reduction

    def test_no_adjustment_when_no_guards(self):
        """No adjustment when no mitigating factors."""
        adjuster = ContextAdjuster()
        context = ContextFactors()  # All false

        adjusted, adjustments = adjuster.adjust(0.8, "vm-001", context)

        assert adjusted == 0.8
        assert len(adjustments) == 0

    def test_access_gate_reduces_auth_pattern(self):
        """Access gate should reduce auth pattern confidence."""
        adjuster = ContextAdjuster()
        context = ContextFactors(has_access_gate=True)

        adjusted, _ = adjuster.adjust(0.8, "auth-001", context)

        assert adjusted < 0.8

    def test_safe_erc20_reduces_token_pattern(self):
        """SafeERC20 should reduce token pattern confidence."""
        adjuster = ContextAdjuster()
        context = ContextFactors(uses_safe_erc20=True)

        adjusted, _ = adjuster.adjust(0.8, "token-001", context)

        assert adjusted < 0.8

    def test_trusted_call_reduces_confidence(self):
        """Trusted external call should reduce confidence."""
        adjuster = ContextAdjuster()
        context = ContextFactors(external_call_targets=["UniswapV3Router"])

        adjusted, _ = adjuster.adjust(0.8, "vm-001", context)

        assert adjusted < 0.8

    def test_min_confidence_floor(self):
        """Should not go below minimum confidence."""
        config = ContextAdjustmentConfig(min_confidence=0.05)
        adjuster = ContextAdjuster(config)
        context = ContextFactors(
            has_reentrancy_guard=True,
            follows_cei_pattern=True,
            uses_openzeppelin=True
        )

        adjusted, _ = adjuster.adjust(0.5, "vm-001", context)

        assert adjusted >= 0.05

    def test_no_over_adjustment(self):
        """Multiple factors should not over-adjust."""
        adjuster = ContextAdjuster()
        context = ContextFactors(
            has_reentrancy_guard=True,
            follows_cei_pattern=True
        )

        adjusted, _ = adjuster.adjust(0.9, "vm-001", context)

        # Should still be meaningful (floor at 5%)
        assert adjusted >= 0.05
        # But significantly reduced
        assert adjusted < 0.3

    def test_adjustments_list_populated(self):
        """Adjustments list should show what was applied."""
        adjuster = ContextAdjuster()
        context = ContextFactors(has_reentrancy_guard=True)

        _, adjustments = adjuster.adjust(0.8, "vm-001", context)

        assert len(adjustments) == 1
        assert "reentrancy_guard" in adjustments[0]

## Common Pitfalls

1. **Pitfall:** Over-aggressive context adjustment makes all scores low
   - **Fix:** Use floor at 5%, monitor distribution after adjustment

2. **Pitfall:** Missing context factors for new patterns
   - **Fix:** Default to no adjustment when unknown

3. **Pitfall:** Trusted contract list outdated
   - **Fix:** Make it configurable, review periodically
