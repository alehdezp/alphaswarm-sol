## Data Sources (Priority Order)

### Source 1: DVDeFi Benchmark (Highest Quality)

**Location:** `benchmarks/dvdefi/`
**Why:** 13 labeled challenges with known vulnerabilities
**Expected yield:** ~50-100 findings

**Extraction Steps:**
1. Run BSKG on each DVDeFi contract
2. Compare findings to known vulnerabilities
3. Label: TP if matches known vuln, FP otherwise
4. Document: contract name, line number, pattern ID

### Source 2: SmartBugs Dataset

**Where to get:** https://github.com/smartbugs/smartbugs
**Why:** Curated vulnerable contracts with labels
**Expected yield:** ~100-200 findings

**Extraction Steps:**
1. Clone SmartBugs (or use subset)
2. Run BSKG on vulnerable contracts
3. Match findings to known vulnerabilities
4. Label systematically

### Source 3: Real Audit Reports

**Location:** `docs/contracts/` (if exists) or external
**Why:** Production-grade ground truth
**Expected yield:** ~30-50 findings

**Extraction Steps:**
1. Find contracts with published audit reports
2. Run BSKG on those contracts
3. Compare to audit findings
4. Label based on auditor findings

### Source 4: Manual Labeling of BSKG Findings

**Location:** Run BSKG on test projects in `tests/projects/`
**Why:** Fill gaps in other sources
**Expected yield:** ~50 findings

**Labeling Protocol:**
1. Two reviewers independently label each finding
2. Disagreements resolved by third reviewer
3. Document reasoning for each label

## Test Requirements

Create: `tests/test_calibration.py`

```python
import pytest
from true_vkg.calibration.dataset import LabeledDataset, LabeledFinding

def test_dataset_loading():
    """Test loading labeled dataset."""
    dataset = LabeledDataset.load("data/labeled/ground_truth.json")
    assert len(dataset.findings) >= 200
    assert all(isinstance(f, LabeledFinding) for f in dataset.findings)

def test_label_distribution():
    """Test label distribution is reasonable."""
    dataset = LabeledDataset.load("data/labeled/ground_truth.json")
    dist = dataset.distribution()
    # Should not be all one class
    assert 0.1 < dist["tp_rate"] < 0.9

def test_train_test_split():
    """Test stratified split."""
    dataset = LabeledDataset.load("data/labeled/ground_truth.json")
    train, test = dataset.train_test_split(0.3)
    assert len(test) / len(dataset.findings) == pytest.approx(0.3, rel=0.1)
```

## Common Pitfalls

1. **Pitfall:** Only including true positives
   - **Fix:** Deliberately run on safe contracts to get FPs

2. **Pitfall:** Labeling based on BSKG output, not ground truth
   - **Fix:** Use external sources (audits, CVEs) for labels

3. **Pitfall:** Unbalanced pattern representation
   - **Fix:** Track distribution, add data for underrepresented patterns

4. **Pitfall:** Label leakage between train/test
   - **Fix:** Split by contract, not by finding
