## Objective

Run BSKG on three different protocol types and calculate detection metrics against audit report ground truth.

## CRITICAL CONTEXT

### Honest Expectations

**VKG will NOT catch everything.** Expect:
- 30-50% recall on pattern-based vulnerabilities
- 60-80% precision (most findings are valid)
- ~0% on business logic bugs (VKG can't understand semantics)

### What We're Actually Testing

1. Does BSKG find reentrancy when it exists?
2. Does BSKG find access control issues?
3. Are VKG's findings accurate (low FP)?
4. What specific patterns does BSKG miss?

### Metrics Definitions

```
True Positive (TP):  BSKG found it, audit report found it
False Positive (FP): BSKG found it, audit report didn't (and it's wrong)
False Negative (FN): BSKG missed it, audit report found it
True Negative (TN):  Neither found it (not measurable)

Precision = TP / (TP + FP)  -- "Of what we found, how much is real?"
Recall    = TP / (TP + FN)  -- "Of what exists, how much did we find?"
F1        = 2 * (P * R) / (P + R)
```

### Step 4: Match Findings (2 hours)

**This is the hard part.** Match BSKG findings to audit findings:

```python
def match_findings(vkg_findings: list, audit_findings: list) -> dict:
    """Match BSKG findings to audit report findings.

    Returns:
        {
            "true_positives": [(vkg, audit), ...],
            "false_positives": [vkg, ...],
            "false_negatives": [audit, ...],
        }
    """
    matched_audit = set()
    result = {
        "true_positives": [],
        "false_positives": [],
        "false_negatives": [],
    }

    for vkg in vkg_findings:
        matched = False
        for audit in audit_findings:
            if is_match(vkg, audit):
                result["true_positives"].append((vkg, audit))
                matched_audit.add(audit["id"])
                matched = True
                break

        if not matched:
            result["false_positives"].append(vkg)

    for audit in audit_findings:
        if audit["id"] not in matched_audit:
            # Only count if BSKG could theoretically detect
            if audit.get("vkg_should_find", True):
                result["false_negatives"].append(audit)

    return result

def is_match(vkg: dict, audit: dict) -> bool:
    """Determine if BSKG finding matches audit finding.

    Match criteria (any of):
    1. Same file + same function + related category
    2. Same file + overlapping line range + same category
    """
    # File match
    vkg_file = Path(vkg["location"]["file"]).name
    audit_file = Path(audit["location"]["file"]).name
    if vkg_file != audit_file:
        return False

    # Function match (if available)
    if vkg.get("location", {}).get("function"):
        if vkg["location"]["function"] == audit["location"].get("function"):
            return True

    # Line range overlap
    vkg_line = vkg.get("location", {}).get("line", 0)
    audit_lines = audit.get("location", {}).get("line_range", [0, 0])
    if audit_lines[0] <= vkg_line <= audit_lines[1]:
        return True

    return False
```

### Step 5: Calculate Metrics (30 min)

```python
def calculate_metrics(result: dict) -> dict:
    tp = len(result["true_positives"])
    fp = len(result["false_positives"])
    fn = len(result["false_negatives"])

    precision = tp / (tp + fp) if (tp + fp) > 0 else 0
    recall = tp / (tp + fn) if (tp + fn) > 0 else 0
    f1 = 2 * precision * recall / (precision + recall) if (precision + recall) > 0 else 0

    return {
        "true_positives": tp,
        "false_positives": fp,
        "false_negatives": fn,
        "precision": round(precision, 3),
        "recall": round(recall, 3),
        "f1_score": round(f1, 3),
    }
```
