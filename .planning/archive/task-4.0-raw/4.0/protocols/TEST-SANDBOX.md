# Test Scaffold Sandbox Security

## Overview

Test scaffolds generated by BSKG execute in a controlled sandbox to prevent:
1. Accidental state changes to real contracts
2. Unintended transactions on mainnet
3. Resource exhaustion from infinite loops
4. Sensitive data exposure

## Execution Modes

### Mode 1: Dry-Run (Default)

```bash
vkg scaffold generate <finding-id>  # Generates scaffold but doesn't execute
vkg scaffold preview <finding-id>   # Shows what would be generated
```

- No actual compilation
- No test execution
- Returns scaffold code as text
- Safe for any environment

### Mode 2: Sandbox Execution

```bash
vkg scaffold run <finding-id> --sandbox
```

- Creates isolated temp directory
- Uses fork mode with anvil/hardhat
- Timeout: 60 seconds default
- Network: local fork only (no mainnet RPC)

### Mode 3: Full Execution (Explicit Opt-In)

```bash
vkg scaffold run <finding-id> --execute --confirm
```

- Requires explicit confirmation
- Logs all actions
- Respects `.env` RPC settings

## Safety Configuration

### Environment Variables

```bash
# Disable all execution (scaffolds only)
VKG_SCAFFOLD_DRYRUN=1

# Restrict RPC endpoints
VKG_ALLOWED_RPC=http://localhost:8545,http://127.0.0.1:8545

# Timeout for scaffold execution
VKG_SCAFFOLD_TIMEOUT=60

# Disable mainnet fork
VKG_NO_MAINNET_FORK=1
```

### Configuration File (.vrs/config.yaml)

```yaml
scaffold:
  default_mode: dryrun  # dryrun | sandbox | execute
  timeout_seconds: 60
  allowed_rpcs:
    - http://localhost:8545
    - http://127.0.0.1:8545
  block_mainnet_fork: true
  require_confirmation: true
```

## Sandbox Implementation

### Directory Isolation

```python
def create_sandbox() -> Path:
    """Create isolated sandbox for test execution."""
    sandbox = tempfile.mkdtemp(prefix="vkg_sandbox_")

    # Copy only necessary files
    # No .env, no private keys, no credentials

    return Path(sandbox)
```

### Network Restrictions

```python
def validate_rpc(rpc_url: str) -> bool:
    """Validate RPC is allowed."""
    BLOCKED_PATTERNS = [
        "mainnet.infura.io",
        "eth-mainnet.alchemyapi.io",
        "mainnet.etherscan.io",
    ]
    return not any(p in rpc_url for p in BLOCKED_PATTERNS)
```

### Timeout Enforcement

```python
def run_with_timeout(cmd: list, timeout: int = 60) -> Result:
    """Execute command with timeout."""
    try:
        return subprocess.run(cmd, timeout=timeout)
    except subprocess.TimeoutExpired:
        return Result(success=False, error="Timeout exceeded")
```

## Security Checklist

Before scaffold execution:

- [ ] Verify dry-run mode for first run
- [ ] Check RPC endpoint is local/fork
- [ ] Confirm no mainnet private keys in scope
- [ ] Set appropriate timeout
- [ ] Review generated test code

## Integration with CI

```yaml
# .github/workflows/scaffold-tests.yml
jobs:
  scaffold-security:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Run scaffold tests in sandbox
        env:
          VKG_SCAFFOLD_DRYRUN: "1"
          VKG_NO_MAINNET_FORK: "1"
        run: |
          uv run vkg scaffold run --all --sandbox
```
